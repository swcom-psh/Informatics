실측 데이터를 활용한 다익스트라(Dijkstra) 알고리즘 경로 탐색 실습

다익스트라(Dijkstra) 알고리즘 개요

샘플 예제(1학년 교무실 → 급식소)를 기준으로 단계별 동작 과정을 직접 확인해보세요.

목적

하나의 출발지에서 다른 모든 지점까지 가장 짧은 거리를 구하는 알고리즘

핵심 전략

미확정 장소 중 가장 가까운 곳을 먼저 탐색하는 탐욕법(Greedy)

실생활 활용

내비게이션, 지도앱의 최단 경로 탐색에 활용되는 핵심 알고리즘

단계별 동작 확인 0 / 7단계

현재 방문 중 확정 완료 미확정 검사 중 간선 최단 경로
[0단계] 초기 상태
다음 버튼을 눌러 알고리즘 동작을 단계별로 확인해보세요.

파이썬 코드로 살펴보기 Python (heapq) 

import heapq

def dijkstra(graph, start):
    # 1. 모든 장소까지 거리 무한대로 초기화 (출발지만 0)
    distances = {node: float('inf') for node in graph}
    distances[start] = 0

    # 2. 우선순위 큐: [(거리, 장소)]
    queue = [(0, start)]

    while queue:
        # 3. 가장 가까운 미방문 장소 선택
        current_dist, current_node = heapq.heappop(queue)

        if distances[current_node] < current_dist:
            continue  # 이미 더 짧은 경로 존재하면 패스

        # 4. 이웃 장소들의 거리 계산
        for neighbor, weight in graph[current_node].items():
            distance = current_dist + weight

            # 5. 더 짧은 경로 발견 시 갱신!
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))

    return distances
