3-2-2. 표준입출력과 파일입출력

컴퓨터와 안전하게 대화하고 데이터를 영원히 간직하는 방법

💡 전원을 끄면 대화 기록이 다 날아간다고?

컴퓨터에게 "안녕?" 하고 키보드로 타이핑해서 화면에 대답이 뜨게 만드는 것은 표준 입출력이라고 합니다. কিন্তু 램(RAM)에서 돌아가는 이 대화는 까만 도스 창을 끄면 영영 사라집니다.

우리가 작성한 일기장, 온도 센서가 측정한 기록들... 이 소중한 정보들을 컴퓨터의 하드웨어 지퍼백(하드디스크)에 안전하게, 영원히 저장하고 꺼내 보는 기술! 그것이 바로 프로그램의 핵심인 파일 입출력(File I/O)입니다.

1. 컴퓨터랑 실시간으로 떠들기: 표준 입출력

키보드로 값을 받아들이는 input()과 검은색 모니터 창(콘솔)에 바로 결과를 뿌려주는 print()가 있습니다. 기본 중의 기본이죠! 

# 컴퓨터야 내 이름을 물어봐 줘!
name = input("너의 이름은 무엇이니? ")

# 친절하게 화면에 띄우기 (표준 출력)
print("반가워!", name, "오늘도 좋은 하루 보내!")

2. 하드디스크에 일기 쓰기: 파일 입출력 3단계 법칙

파일을 다룰 때는 반드시 문 열기 ➔ 볼일 보기 ➔ 문 닫기 3단계를 거칩니다. 화장실을 이용하는 것과 완전히 똑같습니다!

Step 1

Open (열기) open() 함수로 장부를 펼칩니다. f = open('a.txt', 'w')

Step 2

Read/Write (읽기/쓰기) 글씨를 적거나, 적힌 걸 읽습니다. f.write("안녕!")

Step 3

Close (닫기) ⚠️중요 다 썼으면 파일을 닫아야 저장이 완료됩니다! f.close()

🚨 파일 여는 모드(Mode) 3총사

  • 'r' (Read): 읽기 모드 (파일에 적힌 내용을 눈으로 볼 때만 씀) - 파일이 없으면 에러남!
  • 'w' (Write): 쓰기 모드 (새로 작성할 때) - 주의! 기존 내용이 싹 지워지고 처음부터 다시 씀.
  • 'a' (Append): 추가 보드 (이미 적힌 내용 맨 밑에 밑줄 긋고 이어서 추가할 때)

3. 파이썬 마법의 키워드: with

인간은 깜빡하는 동물이라서, 파일을 열고 f.close() 명령어를 적어주는 것을 자꾸 잊어버립니다. (메모리가 펑!). 그래서 알아서 문을 닫아주는 자동문이 존재합니다!

# with를 쓰면 close()를 안 써도 구역이 끝나면 자동으로 파일을 저장하고 닫아줍니다!

with open("my_diary.txt", "w", encoding="utf-8") as f:
    f.write("2026년 4월 1일.\n")
    f.write("오늘은 파이썬 코딩을 배웠다. 정말 신기했다!\n")
    
# 들여쓰기가 끝나는 순간, 무사히 바탕화면에 my_diary.txt가 생성됨.
