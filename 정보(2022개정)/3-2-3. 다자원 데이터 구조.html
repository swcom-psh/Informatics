3-2-3. 다차원 데이터 구조 (리스트)

수많은 데이터를 한 줄짜리 꼬치나 엑셀 표처럼 묶어보자!

💡 한 달 지출액을 한 바구니에 담기

반 친구 30명의 수학 점수를 저장하려면 변수가 30개 있어야 할까요? 아닙니다! 대괄호 [ ] 를 사용해 하나로 묶으면 됩니다.
이렇게 여러 값을 하나로 묶어 관리하는 데이터 구조를 리스트(List)라고 부릅니다.

1. 일차원 리스트와 인덱스 열차

일차원 리스트는 데이터가 일렬로 나열된 기차와 같습니다.

# 1월~6월 지출액 (만원)
spending = [25, 36, 8, 53, 24, 56]
print(type(spending)) # <class 'list'> 가 출력됨

기차 칸 번호: 인덱스(Index)

프로그래밍에서 번째를 셀 때는 무조건 0번방부터 시작합니다!

[0]
25
[1]
36
[2]
8
[3]
53
[4]
24
[5]
56

2. 리스트를 조종하는 마법의 함수들

  • len(리스트명) : 리스트의 전체 길이(원소의 개수)를 셉니다.
  • sum(리스트명) : 안에 있는 숫자를 전부 더해서 합계를 내줍니다.
  • 리스트명.append(원소) : 리스트의 맨 뒷칸에 새로운 데이터를 달아줍니다. (예: spending.append(70))

3. 엑셀 표를 코드로! (이차원 리스트)

데이터가 많아지면 행(Row)과 열(Column)을 지닌 이차원 표를 만듭니다. 이차원 리스트는 '리스트 안의 리스트'로 구현하며, 두 개의 인덱스를 연달아 사용합니다: 리스트명[행][열]

# 월별 수입과 지출 리스트 [[행1], [행2], [행3]]
account = [
    ['1월', 50, 18],     # 0행
    ['2월', 61, 42],     # 1행
    ['3월', 15, 36]      # 2행
]

# 2월(1행)의 수입(1열) 가져오기!
print(account[1][1]) # 결과: 61
주의! len(account)를 하면 행의 개수인 3이 나옵니다. 첫 번째 행의 칸 수를 알고 싶다면 len(account[0])을 써야 합니다.

📝 스스로 점검하기

Q 1

people = [['홍길동', 30, '개발자'], ['김철수', 25, '디자이너']]
일 때, 홍길동의 직업 '개발자' 라는 값을 출력하려면 코드를 어떻게 짜야할까요?

▶ 정답 확인하기

정답: print(people[0][2])
0번째 행의 2번째 열 데이터에 접근해야 합니다. (홍길동-0열, 나이-1열, 직업-2열)
