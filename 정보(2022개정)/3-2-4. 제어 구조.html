프로그램이 위에서부터 아래로 무작정 실행되는 것을 순차 구조라고 합니다. 하지만 살다보면 '조건'에 따라 다른 길을 가거나(선택), '똑같은 일'을 반복(반복)해야 합니다. 이처럼 컴퓨터가 실행하는 순서를 결정해 주는 문법을 제어 구조라고 부릅니다.

1. 만약에~(If): 선택 구조

특정 상황(조건문) 충족 여부에 따라 실행하는 지시가 갈리는 구조입니다.

① if - else 문 (조건이 하나일 때) 
# 탑승 인원이 25명 이하일까?
if people <= 25:
    print('만석이 아닙니다, 여유!')
else:
    print('좌석에 앉을 수 없습니다.')
② if - elif - else 문 (조건이 여러 개일 때) 
if people <= 25:
    print('여유')
elif people <= 40:      # 위의 조건을 통과하지 못한 상태에서 검사!
    print('보통')
else:                   # 모든 조건에 안 맞으면 여기로 옴
    print('혼잡')
※ 파이썬 초보자 실수 1위: 들여쓰기 에러! (IndentationError)
파이썬은 if 나 명령문 뒤에 콜론(:)을 붙인 다음 줄은 반드시 Tab 키나 스페이스바로 "들여쓰기"를 해서 '이만큼이 if문에 속한 영역이야'라고 알려주어야 합니다!

2. 노선을 빙글빙글 (for): 반복 구조

반복할 범위가 정해져 있을 때 사용하는 문법입니다. 버스가 정해진 정류장을 순서대로 도는 것과 똑같습니다.

# 1. 리스트를 이용한 반복!
for station in ['남부병원', '터미널', '민국역']:
    print(station, '정류장입니다. 정차하겠습니다!')

# 2. 범위를 지정하는 range() 함수
for i in range(1, 5): 
    # 1부터 4까지만 반복함! (끝 번호 - 1 까지만 돕니다)
    print(i, '좋은 아침!')
  • range(5) : 0부터 4까지 5번 돕니다.
  • range(1, 5) : 1부터 4까지 4번 돕니다. (뒤 숫자는 미포함)
  • range(2, 10, 3) : 2, 5, 8 (3칸씩 띄워서 반복)

for i in range(1, 4):
이 For 반복문이 실행되면 안쪽에 있는 명령문은 총 몇 번이나 반복되어 실행될까요?

▶ 정답 확인하기

정답: 총 3번 실행된다.
변수 i 에는 1, 2, 3 이 순서대로 들어가고 반복문은 종료됩니다. (끝 번호 4는 범위에 포함되지 않습니다!)
