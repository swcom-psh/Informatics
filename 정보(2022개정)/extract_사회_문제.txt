  기계학습으로 해결할 수 있는 문제의 특징
기계학습으로 해결할 수 있는 문제의 범위는 매우 넓지만, 세상에 존재하는 모든 문제를

기 적합한 문제의 특징을 올바르게 이해해 문제 해결에 적합한 방법을 선택한다면 최적의

1  복잡한 패턴이 있는 문제
사람이 데이터에서 파악하기 어려운 복잡한 패턴을 기계학
습으로 찾아낼 수 있다. 예를 들어 늑대와 개의 모습에는 분
명히 다른 패턴이 존재하지만, 사람이 이를 구분하는 규칙을 
모두 명확하게 제시하기는 어렵다. 이와 같이 사람이 모든 패
턴을 정의할 수 없는 문제는 기계학습으로 해결하기에 적합 
하다.
2  많은 데이터를 수집할 수 있는 문제
기계학습은 충분한 양의 데이터를 기반으로 패턴을 찾는다. 
따라서 기계학습으로 문제를 해결하기 위해서는 데이터를 많
이 수집할 수 있어야 한다. 예를 들어 영화의 흥행 여부를 예측
하는 문제는 과거 영화의 개봉 시기, 관람객 수, 리뷰, 평점 등
의 데이터가 충분하다면 기계학습으로 해결할 수 있다.
3  과거의 데이터를 기반으로 예측하는 문제
기계학습은 수집된 데이터를 기반으로 모델을 학습하여 새
로 입력된 데이터의 결과를 예측한다. 따라서 과거의 데이터
를 기반으로 새로운 사실을 예측하는 문제는 기계학습으로 해
결할 수 있다. 예를 들어 사용자가 ‘좋아요’를 누른 음악 데이터
의 패턴을 학습해 새로 나온 신곡에 ‘좋아요’를 누를지 예측하
는 것은 기계학습으로 해결하기에 적합한 문제이다.

  기계학습의 문제 해결 과정과 인공지능 윤리
기계학습으로 문제를 해결하는 과정은 다음 5단계로 이루어진다. 각 단계는 서로 밀접하
게 연결되어 있으며, 문제에 따라 생략되거나 반복될 수도 있다. 또한, 인공지능 모델이 사
회, 경제, 환경에 영향을 미칠 수 있기 때문에 기계학습을 활용해 문제를 해결할 때는 반드
시 모든 단계에서 윤리적인 측면을 고려해야 한다.
 기계학습의 한계
•간단한 문제는 사람이 알고리즘을 
직접 설계하여 해결하는 것이 더 
효율적이다.
•데이터의 품질이 좋지 않거나 부
족한 경우, 기계학습으로 문제를 
해결하기 어려울 수 있다.
•기계학습 모델이 도출한 결과를 
인간이 이해하고 해석하기 어려운 
경우, 일부 모델의 결과를 설명하
기 어려울 수 있다.
•고도의 기계학습 모델을 구축하는 
비용이 매우 높아 개인이나 작은 
규모의 기업 또는 기관에서는 문
제를 해결하는 데 걸림돌이 될 수 
있다.
 설명 가능성 문제
기계학습 모델이 도출한 결과를 
인간이 이해하고 해석하기 어려울 
경우 모델의 편향과 같은 문제점을 
파악하기 어렵다. 따라서 설명 가능
한 인공지능을 개발하는 것은 매우 
중요한 연구 주제 중 하나이다.
해결하려는 문제를 정의하고, 관련 데이터를 수집한다.
문제 정의와 데이터 수집
학습된 모델을 실제 문제 상황에 적용하여 문제를 해결한다.
모델 적용
모델이 문제를 해결하기에 적합한 성능을 가졌는지 평가하고, 개
선이 필요하다면 데이터를 추가로 학습하거나 새로운 모델을 적
용하는 등의 조치를 취한다.
모델 평가
기계학습의 유형과 유형에 맞는 적합한 알고리즘을 선택하고, 데
