# 🏃‍♂️ 3-1-3 데이터 탐색: 다익스트라 최단 경로 실습 수업 가이드

학생들이 직접 몸으로 데이터를 수집하고, 그 데이터를 통해 알고리즘의 강력함을 체험해 볼 수 있는 1차시(또는 2차시 블록) 수업 가이드입니다. 

---

## 🎯 수업 개요

*   **관련 단원:** 정보 (2022 개정) 3-1-3. 데이터 탐색
*   **선수 학습:** 순차 탐색, 이진 탐색 개념, 파이썬 기초 문법
*   **학습 목표:**
    1.  그래프(Graph) 자료구조의 기본 요소(노드, 간선)를 실생활 모델을 통해 이해한다.
    2.  다익스트라(Dijkstra) 알고리즘 코드의 완벽한 구현보다는, **알고리즘의 동작 개념과 데이터 활용성**에 초점을 맞춘다.
    3.  수집된 원시 데이터(직접 잰 거리)를 프로그램이 처리할 수 있는 형태로 바꾸어 입력하고 결과를 도출할 수 있다.

---

## ⏰ 단계별 진행 가이드 (50분 기준)

### [도입] (10분) : "급식실까지 가장 빠른 길은 어디일까?"
1.  **문제 상황 제시 (동기유발)**
    *   "종이 치면 다들 급식실로 뛰어갑니다. 어떻게 가는 것이 가장 빠를까요?"
    *   학생들이 평소에 이용하는 다양한 경로(본관 통과, 운동장 가로지르기 등)를 이야기하게 합니다.
2.  **한계 인식과 새로운 개념 도입**
    *   "우리가 배웠던 순차 탐색이나 이진 탐색으로 이 복잡한 갈림길의 최단 거리를 알 수 있을까요?" (단순한 1차원 데이터가 아니므로 불가능함을 지도)
    *   점(장소)과 선(길)으로 이루어진 **그래프(Graph)** 데이터 구조를 소개합니다.
    *   네비게이션에서 사용하는 **다익스트라(Dijkstra) 알고리즘** 키워드를 소개하며, 오늘은 우리가 직접 학교의 네비게이션이 되어본다고 안내합니다.

### [전개 1] (20분) : 발로 뛰는 데이터 수집 (야외 활동)
1.  **모둠 구성 및 미션 부여**
    *   3~4인 1조로 구성합니다.
    *   준비물: 줄자 (또는 스마트폰의 기본 거리측정(AR) 앱), 메모장
2.  **거리 데이터 측정**
    *   각 조별로 특정 건물(예: 1학년 1반 교실 앞)에서 급식소까지 이어지는 주요 **거점(노드)** 들을 3~5개 정도 정하게 합니다. (예: 수돗가, 중앙 현관, 도서관 앞 등)
    *   거점과 거점 사이를 잇는 **길(간선)** 의 길이를 직접 측정하여 표나 그림으로 기록해 오게 합니다.

### [전개 2] (15분) : 알고리즘 적용 및 데이터 활용 (HTML 실습)
1.  **웹 프로그램 실행**
    *   교사가 배포한 `3-1-3. 다익스트라 경로 탐색 실습.html` 파일을 웹 브라우저에서 엽니다. (별도 설치 없이 실행됨)
2.  **데이터 입력 (모델링)**
    *   **[장소 추가]**: 야외에서 측정한 거점의 이름(본관, 수돗가 등)을 차례대로 추가합니다.
    *   **[거리 추가]**: 노드와 노드를 선택하고, 측정한 거리(m)를 꼼꼼히 연결합니다. (화면 우측에 나만의 학교 지도가 그려지는 것을 관찰)
3.  **최단 경로 탐색 실행**
    *   출발지와 목표지점(급식소)을 선택 후 **[최단 경로 계산하기]** 버튼을 클릭합니다.
    *   우측 지도에서 형광색으로 하이라이트 된 최단 경로 라인을 확인합니다.

### [정리 및 평가] (5분) : 데이터 활용의 가치
1.  **결과 공유 및 피드백**
    *   조별로 찾아낸 가장 빠른 경로가 어디인지, 총 거리는 얼마인지 발표합니다. 측정 오차에 따라 결과가 다를 수 있음을 이야기합니다.
2.  **핵심 메시지 전달 (가장 중요!)**
    *   "여러분은 오늘 다익스트라라는 복잡한 코드를 직접 짜지 않았습니다. 하지만 여러분이 **직접 수집한 데이터를 알고리즘이 이해할 수 있는 형식으로 잘 정리해서 넣어주기만 해도, 이렇게 훌륭한 문제를 해결할 수 있다는 것**을 배웠습니다."
    *   실제 프로그래머들도 기존에 잘 만들어진 알고리즘(도구)에 데이터를 성공적으로 적용하는 **'데이터 활용 및 문제 해결 능력'** 이 매우 중요함을 강조합니다.
