Chapter 2. 프로그래밍의 기초

2-5-1. 파일의 개념

영구적인 데이터 저장의 기본! 휘발되는 데이터는 이제 안녕!
프로그램이 종료되어도 데이터를 영구적으로 저장하는 방법을 알아봅시다.

Think & Objective

"파일은 어떻게 만들어 사용할 수 있는 것일까? 또 파일의 데이터는 어떻게 저장하며, 저장한 데이터는 어떤 방법으로 불러와 사용하는 것일까?"

  • 파일의 개념과 파일 입출력 과정을 설명할 수 있다.
  • 파이썬을 활용하여 파일을 열고 닫는 방법을 설명할 수 있다.

왜 파일이 필요할까?

Memory vs Storage

파일(File)의 정의

컴퓨터 파일(File)은 일정한 규칙에 따라 저장 장치에 기록한 데이터의 집합입니다. 파이썬은 이 파일을 읽고 쓰는 아주 쉬운 방법을 제공합니다.

  • 변수 (메모리): 프로그램을 끄면 사라집니다. 마치 우리가 공부할 때 잠깐 머릿속에 담아두는 것과 같습니다.
  • 파일 (저장장치): 프로그램을 꺼도 남아있습니다. 공부한 내용을 공책에 적어 보관하는 것과 같습니다.

파일 입출력 과정과 열기 모드

The 3-Step Process & Mode Selection

파일을 다루는 것은 '책을 읽는 과정'과 똑같습니다. 아래의 3단계를 꼭 기억하세요!

1단계
파일 열기

책장에 가서 책 꺼내 펴기

open()
2단계
읽기/쓰기

책 내용 읽거나 필기하기

read() / write()
3단계
파일 닫기

책을 덮고 책장에 넣기

close()

파일 열기 모드(Mode)

파일을 열 때 "어떤 목적으로 열 것인가"를 정해야 합니다.

  • 'r' (읽기, Read): 파일 내용을 읽기만 할 때 사용 (파일이 없으면 에러)
  • 'w' (쓰기, Write): 새로 작성할 때 사용
  • 'a' (추가, Append): 기존 내용 뒤에 이어서 쓰고 싶을 때 사용
주의하세요! 'w' 모드의 함정

open("data.txt", "w")를 실행하는 순간, 만약 data.txt가 이미 존재했다면 그 안의 내용은 깨끗하게 삭제되고 빈 파일이 됩니다. 내용을 보존하면서 뒤에 덧붙이고 싶다면 반드시 'a' 모드를 사용하세요.

가장 기본적인 파일 쓰기 실습

예제 파일 쓰기 (Write)
write_file.py 
# 1. 파일 열기 (이름: hello.txt, 모드: 'w'rite)
f = open("hello.txt", "w")

# 2. 파일에 내용 쓰기
f.write("반갑다, 파일 처리는 처음이지?")

# 3. 파일 닫기 (반드시 닫아야 저장됩니다!)
f.close()
Result

(화면에는 아무것도 출력되지 않지만, 프로그램이 실행된 폴더에 hello.txt 파일이 생성됩니다.)

Tip

마지막에 f.close()를 호출하지 않으면, 파일에 쓴 내용이 디스크에 정상적으로 저장되지 않을 수 있습니다.

파일 내용 가져오기

Reading the Data

이번에는 방금 저장한 파일의 데이터를 프로그램 안으로 다시 불러오는 코드를 작성해 봅시다.

🤔 "선생님, f.close()를 꼭 해야 하나요?"

파일을 열어두고 닫지 않으면, 컴퓨터는 그 파일을 "사용 중"이라고 생각해서 다른 프로그램이 수정하지 못하게 잠가버릴 수 있어요. 또한, 닫지 않고 프로그램이 종료되면 쓴 내용이 저장이 안 될 수도 있답니다. 열었으면 반드시 닫는 습관! 잊지 마세요.

파일 읽기 실습

예제 파일 내용 읽기 (Read)
read_file.py 
# 1. 읽기 모드로 파일 열기
f = open("hello.txt", "r")

# 2. 내용 전체를 읽어 변수에 저장하기
data = f.read()
print(data)

# 3. 닫기
f.close()
Result

반갑다, 파일 처리는 처음이지?

Tip

f.read()는 파일에 있는 모든 내용을 문자열(String) 형태로 한 번에 가져와 반환해 줍니다.

오늘의 핵심 요약 📝

3단계: open → read/write → close
모드 선택: 'r'(읽기), 'w'(덮어쓰기), 'a'(이어쓰기)
