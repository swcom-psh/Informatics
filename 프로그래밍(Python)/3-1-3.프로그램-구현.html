설계도를 꺼내세요!

지난 시간에 우리는 '성적 계산기'의 데이터 입력, 처리, 출력 과정을 구체적으로 설계했습니다. 이제 그 약속된 절차를 선택한 프로그래밍 언어의 문법과 규칙에 맞춰 실제로 입력해 볼 시간입니다.

Python

웹 브라우저에서 실행되는 동적인 앱이나 인터랙티브한 화면을 만들 때 필수입니다.

Java / C++

성능이 중요하거나 하드웨어를 직접 제어하는 대규모 시스템 개발에 적합합니다.

실전! 성적 계산기 구현

설계서에 작성된 알고리즘을 바탕으로, 파이썬을 이용해 실제 동작하는 소스 코드를 작성해 봅시다.

Python Source Code
print("*** 과목별 점수 계산 프로그램 ***")

# 1. 데이터 입력 (설계 단계의 입력을 코드로 구현)
mid_score = float(input("중간고사 점수: "))
fin_score = float(input("기말고사 점수: "))
task_score = float(input("수행평가 점수: "))

# 2. 데이터 처리 (비율 계산 알고리즘 적용)
# 반영 비율: 중간 30%, 기말 30%, 수행 40%
total = (mid_score * 0.3) + (fin_score * 0.3) + (task_score * 0.4)

# 3. 결과 출력 (설계된 형식대로 소수점 첫째 자리까지)
print(f"당신의 최종 점수는 {total:.1f}점입니다.")
재치 있는 구현 팁: 데이터 타입의 중요성

코드에서 float()를 사용하는 이유를 기억하시나요? 정수형(int)만 사용하면 소수점이 포함된 정확한 점수 계산이 불가능합니다. 설계 단계에서 결정한 데이터 타입을 구현에서 지키는 것이 중요합니다!

구현할 때 주의할 점!

컴퓨터는 매우 꼼꼼합니다. 아주 작은 문법적 실수(Syntax Error)도 용납하지 않으니 주의하세요.

  • 오타와 대소문자 구분: 파이썬은 Print()와 print()를 다르게 인식합니다. 반드시 정해진 규칙을 따르세요.
  • 들여쓰기(Indentation): 제어문이나 함수 내에서 들여쓰기를 틀리면 프로그램이 실행되지 않거나 엉뚱하게 동작합니다.
  • 변수 이름 일치: 설계서에서 약속한 변수 이름을 그대로 사용해야 협업 시 혼란을 방지할 수 있습니다.
