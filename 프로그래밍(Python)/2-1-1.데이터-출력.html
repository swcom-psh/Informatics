Chapter 2. 프로그래밍의 기초

2-1-1. 데이터 출력

파이썬이 데이터를 어떻게 구분하는지 배우고, 그 데이터를 화면에 멋지게 출력하는 방법을 마스터해 봅시다.

Think & Objective

"여러 가지 데이터를 컴퓨터로 다루기 위해서는 어떻게 입력하고 출력해야 할까?"

  • 주어진 값을 자료형에 따라 정확하게 표현할 수 있다.
  • 주어진 데이터를 한 줄 또는 여러 줄로 출력할 수 있다.

0. 시작하기 전에: 컴퓨터에게 말 걸기

앞으로 나올 예제 코드들을 보면 항상 print()라는 녀석이 등장할 거예요.

파이썬에서 print()는 "괄호 안의 내용을 화면에 보여줘!"라는 뜻을 가진 마법의 명령어(함수)입니다. 컴퓨터에게 명령을 내리고, 그 결과를 우리 눈으로 확인하기 위해 쓰는 가장 기본적인 확성기랍니다.

지금은 "아, 화면에 결과를 띄워주는 거구나!" 정도로만 생각하고 부담 없이 다음으로 넘어가세요. 자세한 사용법은 아래에서 완벽하게 다룰 거예요!

자료형(Data Type)의 개념과 특징

컴퓨터 내부에서 데이터는 종류와 특성에 따라 저장 및 처리 방식이 다릅니다. 이를 자료형(data type)이라고 합니다.

정수 (Integer)

소수점이 없는 딱 떨어지는 숫자입니다. (예: -1, 0, 100)

"컴퓨터 내부에는 2진수 형태로 저장됩니다."

실수 (Floating-point)

소수점을 포함한 숫자입니다. (예: 3.14, 0.5, -2.0)

"정수와는 다른 특별한 방식으로 저장됩니다."

문자열 (String)

큰따옴표(" ")나 작은따옴표(' ')로 감싼 데이터입니다. (예: "안녕", '123')

논릿값 (Boolean)

참(True) 또는 거짓(False) 중 하나의 값을 가집니다.

왕초보의 흔한 착각: 100과 "100"은 같은 걸까요?

우리 눈에는 둘 다 똑같은 숫자 백(100)으로 보이지만, 컴퓨터에게는 완전히 다른 데이터입니다!

  • 100
    (따옴표 없음) 정수 자료형입니다. 덧셈, 뺄셈 등 수학 계산이 가능한 진짜 숫자입니다.
  • "100"
    (따옴표 있음) 문자열 자료형입니다. 겉모습만 숫자일 뿐, 그림처럼 그려진 글자라서 수학 계산을 할 수 없습니다.

자료형 연산 실습 예제

파이썬의 연산 규칙 (교과서 핵심)

  • 숫자 + 숫자: 두 숫자를 더해 새로운 숫자가 됩니다. 예: 2 + 3 → 5
  • 숫자 + 문자열: 자료형이 달라 연산이 불가능하며 오류가 발생합니다. 예: 1 + "1"
  • 문자열 + 문자열: 두 문자열이 이어붙여집니다. 예: "안녕" + "하세요" → "안녕하세요"
  • 문자열 * 숫자: 문자열을 숫자만큼 반복합니다. 예: "hi" * 3 → "hihihi"
예제 숫자와 문자열의 덧셈
45-1.py 
print(1 + "1")
Error

TypeError: unsupported operand type(s)

숫자와 문자는 서로 더할 수 없습니다.

Tip

파이썬에서 연산을 할 때는 항상 자료형이 같아야 합니다. 숫자와 문자를 더하려면 먼저 숫자를 문자열로 바꾸거나, 문자열을 숫자로 변환해야 합니다. 예: str(1) + "1" → "11"

예제 문자열 반복
45-2.py 
print(3 * "hello")
print(5 * "0")
Result

hellohellohello
00000

Tip

문자를 정수와 곱하면, 해당 문자열이 그만큼 반복됩니다. 반복 횟수는 항상 숫자여야 합니다.

자료형 변환 함수

함수명 기능 설명 예시
int(x) x를 정수 자료형으로 변환 int("10") → 10
float(x) x를 실수 자료형으로 변환 float("3.14") → 3.14
str(x) x를 문자열 자료형으로 변환 str(100) → "100"
chr(x) / ord(x) 유니코드 정수와 문자 간 변환 chr(65) → 'A'

잠깐! '유니코드(Unicode)'가 대체 뭔가요?

표 마지막에 있는 chr()과 ord()를 보고 당황하셨나요?

사실 컴퓨터는 'A', 'B', '가', '나' 같은 글자를 있는 그대로 이해하지 못하고 모든 것을 숫자로만 기억합니다. 그래서 "문자 'A'는 숫자 65로 기억하자!"라고 전 세계 사람들이 미리 약속해 둔 규칙표가 있는데, 이 거대한 규칙표를 바로 '유니코드'라고 부릅니다.

  • ord("A") : "문자 'A'는 컴퓨터 안에서 무슨 숫자야?" → 65를 알려줌
  • chr(65) : "숫자 65번은 규칙표에서 무슨 글자야?" → 'A'를 알려줌

지금 당장 이 함수들을 외울 필요는 없어요! "아~ 컴퓨터는 글자도 비밀번호 같은 숫자로 바꿔서 기억하는구나!" 정도로만 이해하고 넘어가도 아주 훌륭합니다.

자료형 변환의 실제 활용 (에러 해결하기)

앞선 자료형(Data Type)의 개념과 특징 섹션에서 1 + "1"처럼 정수와 문자열을 바로 더하려고 하면 TypeError(자료형 오류)가 발생한다는 것을 확인했습니다. 컴퓨터는 서로 다른 종류의 데이터를 어떻게 섞어야 할지 모르기 때문입니다.

이럴 때 바로 방금 배운 자료형 변환 함수를 사용합니다! 데이터의 종류를 하나로 통일해주면 에러 없이 안전하게 연산과 출력을 수행할 수 있습니다. 독학할 때 가장 많이 겪는 에러 중 하나이니 아래 예제를 통해 확실히 원리를 이해해 봅시다.

해결 공식

  • 숫자로 계산하고 싶을 때: 문자열을 정수나 실수로 변환 → 1 + int("1")
  • 문자로 이어 붙이고 싶을 때: 숫자를 문자열로 변환 → str(1) + "1"

형변환을 통한 에러 해결 실습

예제 문자를 숫자로 바꿔서 덧셈하기
convert-1.py 
print(int("100") + 50)
Result

150

Tip

"100"이 진짜 숫자 100이 되었기 때문에 정상적으로 덧셈(수학 연산)이 수행됩니다.

예제 숫자를 문자로 바꿔서 이어 붙이기
convert-2.py 
print(str(100) + "점")
Result

100점

Tip

숫자 100이 문자 "100"으로 변해서 뒤의 "점"이라는 글자와 자연스럽게 합쳐집니다.

화면 출력 (print) 마스터하기

01

기본 출력과 쉼표(,) 활용

print() 함수 내에서 값을 쉼표로 나열하면 공백(스페이스)을 사이에 두고 순서대로 출력됩니다.

예제 기본 데이터 출력
47.py 
print("Hello")
Result

Hello

예제 쉼표를 이용한 나열
48.py 
print("Hello", "Python", "World")
Result

Hello Python World

02

구분자 설정 (sep 옵션)

데이터 사이에 공백 대신 특정 문자를 넣고 싶을 때 sep 옵션을 사용합니다.
여기서 sep은 separator(구분자)의 약자로, 출력할 각 값 사이에 넣고 싶은 문자를 지정할 수 있습니다. 기본 구분자는 공백(" ")입니다.

예를 들어, print(5, 4, 6, "안녕")을 실행하면 기본 구분자인 공백으로 값들이 나뉘어 5 4 6 안녕과 같이 출력됩니다. 점(.)으로 구분하고 싶으면 sep='.'를 사용하면 됩니다.

예제 구분자 지정 출력
49.py 
print("3", "01", "004", sep=".")
Result

3.01.004

03

줄 바꿈 제어 (end 옵션)

print() 함수는 기본적으로 출력이 끝나면 자동으로 한 줄 내려갑니다. 즉, 각 print 문 끝에는 줄바꿈 문자("\n")가 붙습니다.

줄을 바꾸지 않고 이어서 출력하려면 end 옵션을 사용합니다. 예: print("안녕하세요", end=""). 줄바꿈을 원하면 end="\n"을 사용합니다.

기본 줄바꿈

print("안녕하세요")
print("여러분")
                

안녕하세요
여러분

print()를 여러 번 쓰면 각 출력이 새 줄에서 시작합니다. 즉, 자동 줄바꿈(\n)이 추가됩니다.

줄바꿈 없이 이어서 출력 (end 옵션)

print("안녕하세요", end="")
print("여러분")
                

안녕하세요여러분

end=""를 사용하면 다음 print()가 같은 줄에서 이어집니다. 줄바꿈을 원하면 end="\n"을 쓰면 됩니다.

예제 기본 줄 바꿈 동작
50.py 
print("Hello")
print("World")
Result

Hello
World

예제 줄 바꿈 없이 출력
51.py 
print("Hello", end=", ")
print("Python")
Result

Hello, Python

예제 sep와 end 사용
print_sep_end.py 
print(1, 0, 20, sep="")
print("a", "b", end="")
print("c")
Result

1020
a bc

예제 end로 여러 줄 바꿈
print_newline.py 
print(100, 200, 300, sep="", end="\n\n\n")
print('a', 'b', 'c')
Result

100200300


a b c

학습 확인 (형성 평가)

Q1. 다음 중 정수(int) 자료형인 것을 모두 고르시오.

① "123"
② -5
③ 3.14
④ 0
정답 및 해설 보기
정답: ②, ④
① "123"은 큰따옴표로 묶여 있으므로 문자열(str)입니다. 겉보기엔 숫자 같지만 컴퓨터는 글자로 인식합니다.
③ 3.14는 소수점이 있으므로 실수(float)입니다.
따라서 소수점이 없는 순수한 숫자인 ② -5와 ④ 0이 정수입니다.

Q2. print(10, 20, sep=":") 의 실행 결과는?

정답 및 해설 보기
정답 (실행 결과)
10:20
sep=":" 옵션은 쉼표로 나열된 데이터들(10과 20) 사이에 공백 대신 콜론(:)을 집어넣으라는 명령어입니다. 따라서 10과 20이 콜론으로 연결되어 출력됩니다.

데이터 출력 핵심 요약

자료형 (int, float, str, bool)
sep (구분 문자)
end (줄바꿈 제어)
