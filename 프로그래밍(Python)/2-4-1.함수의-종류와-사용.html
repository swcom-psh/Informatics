Chapter 2. 프로그래밍의 기초

2-4-1. 함수의 종류와 사용

파이썬이 기본으로 제공하는 내장 함수부터 다양한 모듈(라이브러리)을 불러와 활용하는 방법까지 함수 사용의 기초를 알아봅니다.

Think & Objective

"함수는 어떻게 만들어졌으며, 내가 원하는 작업을 수행하는 함수는 어떻게 만들 수 있을까?"

  • 함수의 개념, 종류, 필요성 등을 설명할 수 있다.
  • 파이썬 내장 함수를 활용하여 다양한 코드를 작성할 수 있다.

내장 함수 (Built-in Functions)

파이썬을 설치하면 별도의 설정 없이 바로 사용할 수 있도록 기본적으로 내장된 함수들을 의미합니다.
지금까지 우리가 사용했던 print()나 input(), int() 등이 모두 내장 함수입니다.

내장 함수 실습 예제

예제 길이를 구하는 함수 len()
129.py 
s = input()
print(len(s))
설명

len() 함수는 문자열의 길이(글자 수)나 리스트의 데이터 개수를 반환해 주는 매우 유용한 내장 함수입니다.

모듈(Module)과 외장 함수

기본 내장 함수만으로는 한계가 있습니다. 수학, 통계, 그래픽 등 특정 분야를 위해 미리 만들어둔 수많은 함수들의 묶음을 모듈(Module)이라고 합니다. 모듈 안의 함수를 쓰려면 먼저 import 키워드로 모듈을 불러와야 합니다.

모듈 가져오기 (import) 실습 예제

예제 모듈 전체를 그대로 가져오기
130.py 
import math

a, b = map(int, input().split())
print(math.gcd(a, b))
설명

import math를 통해 수학 모듈을 통째로 가져옵니다. 사용할 때는 반드시 math.gcd() 처럼 모듈이름.함수이름 형태로 소속을 명시해야 합니다. (gcd는 최대공약수를 구하는 함수입니다)

예제 모듈에 짧은 별칭 붙여 가져오기 (as)
131_1.py 
import math as mt

a, b = map(int, input().split( ))
print(mt.gcd(a, b))
설명

as mt를 붙이면 math라는 긴 이름 대신 mt라는 짧은 별명으로 모듈의 함수를 호출할 수 있어 코딩이 한결 편해집니다.

예제 모듈 이름 없이 함수만 가져오기 (from import)
131_2.py 
from math import *

a, b = map(int, input().split( ))
print(gcd(a, b))
설명

from math import * 구문을 사용하면 math 모듈에 있는 모든 기능(*)을 불러옵니다. 이때는 앞에 모듈 이름(math.)을 붙일 필요 없이 내장 함수처럼 gcd()만 단독으로 쓸 수 있습니다.

학습 정리

내장 함수 (Built-in)

별도의 선언이나 불러오기 과정 없이 print, input, len 처럼 곧바로 이름만 적어 사용할 수 있는 파이썬 기본 제공 함수들입니다.

모듈 가져오기 (import)

  • • import 모듈 : 정석적인 방법. 모듈명.함수() 형태.
  • • import 모듈 as 약어 : 모듈명에 별칭을 붙여 편하게 씁니다.
  • • from 모듈 import * : 함수명만 단독으로 바로 씁니다.
