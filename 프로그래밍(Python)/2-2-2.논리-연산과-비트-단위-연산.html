Chapter 2. 프로그래밍의 기초

2-2-2. 논리 연산과 비트 단위 연산

참과 거짓을 다루는 논리 연산의 진리표를 이해하고, 데이터의 최소 단위인 비트(Bit)를 직접 제어하는 연산자를 배웁니다.

Think & Objective

"컴퓨터에 저장한 데이터는 어떻게 계산하고 처리할 수 있을까?"

  • 논리 연산자를 사용할 수 있다.
  • 비트 단위 연산자를 사용할 수 있다.

논리(Logical) 연산자

논리 연산자는 참(True)과 거짓(False) 상태를 결합하거나 반전시킬 때 사용합니다. 주로 여러 조건을 동시에 판단해야 할 때 쓰입니다.

연산자 의미 설명
not A 부정 (NOT) A가 참이면 거짓, 거짓이면 참으로 바꿉니다. (반전)
A and B 논리곱 (AND) A와 B가 모두 참일 때만 결과가 참이 됩니다.
A or B 논리합 (OR) A와 B 둘 중 하나라도 참이면 결과가 참이 됩니다.

논리 연산 진리표 (Truth Table)

A B not A A and B A or B
True True False True True
True False False False True
False True True False True
False False True False False

논리 연산 실습 예제

예제 not 연산
80.py 
print(not (123<456))
설명

123<456은 True지만, not으로 반전되어 False가 출력됩니다.

예제 입력 비교 부정
81.py 
a = input()
b = input()
a = int(a)
b = int(b)
print(not (a==b))
설명

a와 b가 다를 때 최종적으로 True가 됩니다.

예제 or 연산
82.py 
n = int(input())
print((n%2==0) or (n%3==0))
설명

입력된 수 n이 2의 배수 '이거나' 3의 배수이면 True입니다.

예제 and 연산
83.py 
n = int(input())
print((n%2==0) and (n%3==0))
설명

입력된 수 n이 2의 배수 '이면서' 3의 배수(즉, 6의 배수)일 때만 True입니다.

비트 단위(Bitwise) 연산자

컴퓨터는 데이터를 0과 1의 비트(Bit) 형태로 저장합니다. 비트 단위 연산자는 이러한 이진수의 각 자리(비트)끼리 직접 연산을 수행합니다.

연산자 이름 설명
~ 비트 NOT 모든 비트를 반전시킵니다. (0은 1로, 1은 0으로)
& 비트 AND 두 비트가 모두 1일 때만 1이 됩니다.
| 비트 OR 두 비트 중 하나라도 1이면 1이 됩니다.
^ 비트 XOR 두 비트가 서로 다를 때만 1이 됩니다. (배타적 논리합)
<< 왼쪽 시프트 비트를 왼쪽으로 이동시킵니다. (이동 횟수만큼 2를 곱하는 효과)
>> 오른쪽 시프트 비트를 오른쪽으로 이동시킵니다. (이동 횟수만큼 2로 나누는 효과)

비트 단위 연산 실습 예제

예제 비트 NOT
85.py 
n = int(input())
print(~n)
설명

입력된 정수 n의 모든 비트를 반전합니다. (파이썬에서는 -n-1 의 결과가 나옵니다.)

예제 비트 OR
86.py 
a = int(input())
b = int(input())
print(a|b)
설명

두 값의 각 비트 자리를 비교하여 하나라도 1이면 1이 됩니다.

예제 비트 AND
88.py 
a = int(input())
b = int(input())
print(a&b)
설명

두 값의 비트가 모두 1일 때만 해당 자리의 결과가 1이 됩니다.

예제 비트 XOR
89.py 
a = int(input())
b = int(input())
print(a^b)
설명

두 값의 각 비트가 서로 다를 경우에만 1이 됩니다.

예제 왼쪽 시프트 연산
91.py 
n = int(input())
k = int(input())
print(n<<k)
설명

정수 n의 비트를 왼쪽으로 k번 이동시킵니다. 이는 n에 $2^k$을 곱하는 것과 똑같은 결과 및 성능 최적화 효과를 가집니다.

예제 오른쪽 시프트 연산
92.py 
n = int(input())
k = int(input())
print(n>>k)
설명

정수 n의 비트를 오른쪽으로 k번 이동시킵니다. 이는 n을 $2^k$으로 나눈 몫을 구하는 것과 똑같은 효과를 가집니다.

학습 정리

논리 연산의 핵심

and는 모든 조건이 엄격하게 참이어야 할 때, or는 하나의 조건만 만족해도 통과시켜야 할 때 사용합니다.

비트 연산의 쓰임새

  • • &, |, ^: 하드웨어 제어 및 권한 관리(마스킹)
  • • <<, >>: 매우 빠른 곱셈/나눗셈 최적화
