Chapter 2. 프로그래밍의 기초

2-3-1. 선택 구조

상황에 따라 다르게 행동하기
프로그램이 주어진 조건에 따라 스스로 판단하고 다른 결과를 내도록 만드는 if, else 문의 기초를 배웁니다.

Think & Objective

"어떤 기준값이나 상태에 따라서 선택적으로 실행하거나 반복하는 구조는 어떻게 만들 수 있을까?"

  • 선택 구조를 사용할 수 있다.
  • 선택 구조의 중첩을 활용하여 복잡한 논리를 표현할 수 있다.

선택 구조의 개념

프로그램 실행 중 특정 조건이 참(True)인지 거짓(False)인지에 따라 실행할 명령을 결정하는 구조입니다. 마치 길을 가다가 갈림길에서 표지판을 보고 방향을 정하는 것과 같습니다.

필수 문법: 들여쓰기(Indentation)

파이썬에서 선택 구조를 작성할 때 가장 중요한 것은 들여쓰기입니다.

  • 조건문 끝에는 반드시 콜론(:)을 붙입니다.
  • 조건에 해당할 때 실행할 문장은 반드시 4칸(또는 Tab)을 들여써야 합니다.
  • 들여쓰기가 일치하지 않으면 IndentationError가 발생합니다.

단순 if 구조

if문은 주어진 조건이 참(True)일 때만 특정 코드를 실행합니다. 조건이 거짓이면 아무 일도 일어나지 않고 넘어갑니다.

기본 형식

if 조건식:
    명령문 # 조건이 참일 때 실행

조건식이 참(True)일 때만 명령을 수행하고, 거짓이면 아무것도 하지 않고 다음으로 넘어갑니다.

파이썬의 핵심: 들여쓰기 (Indentation)

파이썬에서는 콜론(:) 뒤에 오는 코드는 반드시 들여쓰기(스페이스 4칸 또는 Tab)를 해야 if문에 속한 것으로 인식합니다.

if 조건식:
----실행할 명령 1 # 반드시 들여쓰기 필요
----실행할 명령 2

단순 if 실습 예제

예제 기본 if문

123 < 456은 참(True)이므로 들여쓰기 된 print("Hello")가 실행됩니다.

99_1.py 
if 123<456:
  print("Hello")
예제 True 조건

조건 자리에 직접 True를 적었으므로 안의 코드가 무조건 실행됩니다.

99_2.py 
if True:
  print("Python")
예제 False 조건

조건이 False이므로 내부 코드는 무시되고 아무것도 출력되지 않습니다.

99_3.py 
if False:
  print("World")
예제 절댓값 구하기

입력된 수 n이 0보다 작을 때(음수)만 부호를 바꿔 양수로 만듭니다.

100.py 
n = int(input())

if n<0:
  n = -n

print(n)
예제 두 수 중 큰 수 출력

단순 if문 두 개를 사용하여, a가 크면 a를 출력하고 b가 크면 b를 출력합니다.

101.py 
a = int(input())
b = int(input())

if a>=b:
  print(a)

if a<b:
  print(b)

if-else 구조

else를 사용하면 조건이 거짓(False)일 때 실행할 코드를 따로 지정할 수 있습니다. "이거면 이렇게 하고, 아니면 저렇게 해라"라는 논리를 만듭니다.

기본 형식

if 조건식:
    명령문1 # 조건식이 참일 때
else:
    명령문2 # 조건식이 거짓일 때

문법 포인트: '아니면'의 처리

조건이 참인 경우와 거짓인 경우 두 가지 갈래를 모두 지정하고 싶을 때 사용합니다. else 뒤에는 절대로 조건식을 쓰지 않는다는 점에 주의하세요!

if-else 실습 예제

예제 기본 if-else

123 > 456은 거짓이므로 if 안의 코드는 건너뛰고 else 안의 World가 출력됩니다.

102.py 
if 123>456:
  print("Hello")
else:
  print("World")
예제 짝수/홀수 판별

2로 나눈 나머지(% 연산자)가 0이면 짝수(even), 아니면 홀수(odd)로 판별합니다.

103.py 
n = int(input())

if (n%2)==0:
  print("even")
else:
  print("odd")
예제 두 수의 차이 구하기

두 수의 차이를 무조건 양수로 나오게 하기 위해, 큰 수에서 작은 수를 빼도록 구성합니다.

104.py 
a = int(input())
b = int(input())

if a>=b:
  print(a-b)
else:
  print(b-a)

다중 조건과 중첩 if

경우의 수가 3가지 이상일 때는 if문 안에 또 다른 if문을 넣는 중첩(Nested) 구조를 만들거나, 논리 연산자(and, or)를 활용하여 여러 개의 독립적인 if문을 조합할 수 있습니다.

기본 형식 

if 조건식1:
    명령문1 # 조건식1이 참일 때
else:
    if 조건식2:
        명령문2  # 조건식1이 거짓이고, 동시에 조건식2가 참일 때
    else:
        명령문3  # 조건식1이 거짓이고, 조건식2도 거짓일 때
if 조건식1:
    명령문1 # 조건식1이 참일 때
elif 조건식2:
    명령문2  # 조건식1이 거짓이고, 동시에 조건식2가 참일 때
else:
    명령문3  # 조건식1이 거짓이고, 조건식2도 거짓일 때

중첩 및 다중 조건 실습 예제

예제 중첩 if로 성적 판별

먼저 80 이상인지 검사하여 아니면(else) 그 안에서 다시 60 이상인지를 검사하는 이중 구조입니다. 들여쓰기가 계속 깊어지는 것을 확인하세요.

105.py 
n = int(input())

if n>=80:
  print("A")
else:
  if n>=60:
    print("B")
  else:
    print("C")
예제 독립된 if문과 and 연산

예제 105와 결과는 같습니다. 하지만 중첩 없이 3개의 독립된 if문과 and 연산자를 활용하여 조건을 직관적으로 나열했습니다.

106.py 
n = int(input())

if n>=80:
  print("A")

if n<80 and n>=60:
  print("B")

if n<60:
  print("C")
예제 세 수 중 가장 큰 수 찾기

a, b, c 세 개의 수를 입력받아, 먼저 a와 b를 비교하고 그 결과(참/거짓)에 따라 다시 c와 비교하는 깊은 중첩 구조를 보여줍니다.

107.py 
a = int(input())
b = int(input())
c = int(input())

if a>=b:
  if a>=c:
    print(a)
  else:
    print(c)
else:
  if b>=c:
    print(b)
  else:
    print(c)

핵심 요약

if (만약에)
elif (아니면 만약)
else (그 외에)
