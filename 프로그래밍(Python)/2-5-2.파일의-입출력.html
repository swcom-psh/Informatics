2-5-2. 파일의 입력과 출력

화면 출력을 넘어, 실제 파일(TXT, CSV)을 생성하여 데이터를 영구적으로 저장하고 읽어오는 방법을 완벽하게 마스터해 봅시다.

Think & Objective

"파일은 어떻게 만들어 사용할 수 있는 것일까? 또 파일의 데이터는 어떻게 저장하며, 저장한 데이터는 어떤 방법으로 불러와 사용하는 것일까?"

  • 파일을 사용해서 데이터를 읽고 쓸 수 있다.
  • 파이썬으로 TXT 파일과 CSV 파일을 열고 쓸 수 있다.

텍스트(TXT) 파일에서 데이터 읽기

파일을 읽어올 때는 모드를 "r" (Read)로 지정합니다.

  • read() : 파일의 전체 내용을 하나의 긴 문자열로 모두 읽어옵니다.
  • split("\n") : 줄바꿈 문자를 기준으로 문자열을 잘라 리스트(List)로 만들어주면, 한 줄씩 반복 처리하기 매우 좋습니다.
준비 읽어올 파일 1 (158in.txt)

hello
python
world
good

예제 파일을 읽고 줄별로 출력하기
158.py 
f = open("in.txt", "r")
rows = (f.read()).split("\n")
f.close()

for row in rows:
  print(row)
설명

전체 데이터를 읽은 후 \n 단위로 쪼개어 리스트를 만들고 파일을 닫았습니다. 이후 for문을 돌려 리스트 안의 데이터를 한 줄씩 깔끔하게 꺼내 화면에 출력합니다.

준비 읽어올 파일 2 (159in.txt)

2
13
167
1009

예제 입력받은 줄 수만큼 데이터 읽기
159.py 
f = open("in.txt", "r")
rows = (f.read()).split("\n")
f.close()

n = int(input())

for row in rows[:n]:
  print(int(row))
설명

리스트 슬라이싱 rows[:n]을 활용하여 위에서부터 원하는 n개의 줄만 꺼내어 반복합니다. 읽어온 글자를 연산 가능한 숫자 int()로 변환하여 출력합니다.

텍스트(TXT) 파일에 데이터 쓰기

파일을 다루는 기본 절차는 1.열기(open) → 2.작업(read/write) → 3.닫기(close) 순서입니다.

  • open("파일명", "w") : 파일을 엽니다. "w" 모드는 쓰기(Write) 전용 모드입니다. 파일이 없으면 새로 만듭니다.
  • write("내용") : 파일에 내용을 기록합니다. 줄바꿈이 자동으로 되지 않으므로 \n을 사용해야 합니다.
  • close() : 파일 작업을 마치면 반드시 닫아주어야 변경 사항이 안전하게 저장됩니다.

파일 쓰기 실습 예제

예제 기본적인 파일 쓰기
155.py 
f = open("out.txt", "w")

f.write("hello\n")
f.write("world\n")

f.close()
생성된 파일 (out.txt)

hello
world

예제 반복문과 사용자 입력을 결합한 쓰기
156.py 
f = open("out.txt", "w")

n = int(input())

for i in range(n):
  d = int(input())
  f.write(str(d))
  f.write("\n")

f.close()
설명 및 결과

숫자를 파일에 쓸 때는 str()을 이용해 반드시 문자로 변환해야 오류가 나지 않습니다. 만약 입력으로 3, 그리고 차례로 123, 456, 789를 넣었다면 아래와 같은 파일이 만들어집니다.

123
456
789

CSV 파일 읽어오기

CSV (Comma-Separated Values)는 각 데이터가 쉼표(,)로 구분된 텍스트 파일입니다.
엑셀이나 공공 데이터 포털 등에서 데이터를 다룰 때 가장 흔하게 쓰이는 데이터 규격입니다.

개념 CSV 데이터 구조 살펴보기 (162datax.csv 예시)

실제 저장된 형태 (메모장 등)

1,5,4,2
4,6,9,1
0,4,1,2
9,5,7,4

우리가 인식하는 형태 (표)

1 5 4 2
4 6 9 1
0 4 1 2
9 5 7 4

CSV 읽기 및 파싱 실습 예제

준비 읽어올 CSV (161data.csv)

alfa,bravo,charlie,delta,echo,foxtrot,golf,hotel 0.435,0.335,0.11,0.334,0.1355,0.0775,0.0965,7 0.585,0.45,0.125,0.874,0.3545,0.2075,0.225,6 ...

예제 CSV 읽고 쉼표 단위로 분리하기
161.py 
f = open("data.csv", "r", encoding="utf-8-sig")
rows = (f.read()).split("\n")
f.close()

for row in rows:
  values = row.split(",")
  for value in values:
    print(value, end=" ")
  print()
핵심 원리

1. read().split('\n')으로 줄 단위로 쪼갭니다.
2. 각 줄(row)을 다시 split(',')을 적용해 셀 단위(values)로 쪼갭니다.
※ encoding="utf-8-sig"는 한글이나 특수문자가 포함된 CSV 파일을 깨짐 없이 읽어오는 옵션입니다.

CSV 파일로 저장하기

파이썬 내부에서 연산한 2차원 리스트 형태의 데이터를 엑셀 등에서 열어볼 수 있는 CSV 파일로 저장하려면, 데이터 사이에 쉼표(,)를 끼워 넣고 각 행의 마지막에는 줄바꿈(\n)을 넣어주면 됩니다.

예제 2차원 리스트를 CSV로 저장하기
163.py 
rows = [["data A","data B","sum","mean"],
        [1,2,3,1.5],
        [3,4,7,3.5],
        [5,6,11,5.5]]

f = open("dataout.csv", "w")

for row in rows:
  for value in row[:-1]:
    f.write(str(value)+",")
  f.write(str(row[-1])+"\n")

f.close()
쓰기 로직 설명

row[:-1]을 통해 각 행의 마지막 값을 제외한 모든 값에는 값 + "," 형태로 쉼표를 붙여 기록합니다. 마지막 값 row[-1]에는 쉼표 대신 줄바꿈 "\n"을 붙여 다음 행으로 넘어갈 수 있도록 처리합니다.

생성된 파일 (163dataout.csv)

data A,data B,sum,mean
1,2,3,1.5
3,4,7,3.5
5,6,11,5.5

참고 다양한 응용 출력 결과 예시 (164dataoutx.csv)

num,data1,data2
1,79,73
2,83,67
3,97,59
4,71,89

이처럼 파이썬을 활용하면 수천, 수만 개의 원본 데이터를 가공한 후 새로운 형식의 표 데이터(CSV)로 자유자재로 재조립하여 엑셀이나 머신러닝에 활용할 수 있습니다.

핵심 요약

파일 제어의 3단계

항상 open() 으로 열고, read() / write() 로 작업한 뒤, 마지막에 메모리 누수 방지를 위해 close() 로 닫는 습관을 들이세요.

데이터 정제 (Parsing)

파일에서 불러온 모든 것은 '문자열'입니다. split()을 2번 활용해 줄바꿈과 쉼표 기준으로 자르면 2차원 리스트로 깔끔하게 정리할 수 있습니다.
