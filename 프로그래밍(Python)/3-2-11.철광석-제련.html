철광석 제련 알고리즘

2차원 격자에서 특정 범위의 합을 계산하여 가장 가치 있는 구역 탐색하기

슬라이딩 윈도우로 최적 구역 찾기

5×5 광산 격자에서 3×3 제련기를 어디에 놓아야 가장 많은 철광석을 캘 수 있을까요? 이 문제는 2차원 리스트 순회와 슬라이딩 윈도우 기법의 핵심입니다. 가능한 모든 3×3 위치의 합을 계산하고 최댓값을 추적합니다.

Step 1. 프로그램 기획 (문제 정의)

5×5 크기의 광산 격자에 각 위치별 철광석의 가치가 기록되어 있습니다. 3×3 크기의 제련기를 사용하여 한 번에 채굴할 수 있는 영역 중 가장 가치가 높은 합계를 찾아내는 프로그램을 개발합니다.

🔴 원본 오류 수정 — 강조 위치 오류

원본에서 노란색 강조 영역이 (0,0) 위치를 최적 구역인 것처럼 보여줬으나, 실제 계산 결과 (0,0) 합계는 249이고 실제 최대값은 (0,1) 위치의 260입니다. 아래 격자에서 올바른 위치(노란색)와 원본 오류 위치(분홍색)를 확인하세요.

입력 조건

5줄에 걸쳐 각 줄마다 공백으로 구분된 5개의 정수가 입력됩니다.

출력 조건

3×3 영역 합계의 최댓값을 출력합니다.
위 예시의 경우: 260

Step 2. 프로그램 설계 (알고리즘)

핵심 로직: 슬라이딩 윈도우 & 4중 반복문

2차원 리스트를 순회하며 3×3 영역의 합을 각각 구하고, 그 중 최댓값을 갱신합니다.

탐색 범위 공식 — 왜 range(3)인가요?

3×3 윈도우가 5×5 격자를 벗어나지 않으면서 이동할 수 있는 시작점의 범위를 계산합니다.

유효한 시작점 수= 격자 크기 (5)− 윈도우 크기 (3)+1 =3
시작점 i, j ∈ {0, 1, 2}→ range(3) = [0, 1, 2]
i=2일 때 윈도우 행 범위: 2, 3, 4 — 마지막 행 인덱스 4 = 5-1 ✅ 경계 이탈 없음

4중 반복문 — 각 루프의 역할

루프 순서 변수 범위 역할
1번째 (가장 바깥) i range(3) → 0,1,2 윈도우 시작점의 행(row) 위치
2번째 j range(3) → 0,1,2 윈도우 시작점의 열(col) 위치
3번째 r range(i, i+3) 3×3 윈도우 내부 행 순회
4번째 (가장 안쪽) c range(j, j+3) 3×3 윈도우 내부 열 순회 + 누적 합산
1. 데이터 구조

5×5 격자 데이터를 저장하기 위한 2차원 리스트를 생성합니다.

2. 탐색 범위

3×3 제련기가 경계를 벗어나지 않도록 시작점(i, j)을 0~2로 제한합니다. (5-3+1=3)

3. 최댓값 비교

if current_sum > max_total:
max_total = current_sum

Step 3. 프로그램 구현 (Python)

4중 for문을 사용하여 격자의 부분합을 계산합니다.

⚠️ max_total = 0 초기화 주의

원본의 max_total = 0은 모든 격자값이 양수라면 문제 없습니다. 그러나 음수 값이 포함된 경우 실제 최대 합이 음수여도 0이 잘못 출력됩니다.

# ① 양수 전용 (이 문제에서 안전)
max_total = 0

# ② 음수 포함 범용 (권장)
max_total = float('-inf')
Python Solution
# 1. 5x5 격자 입력 (리스트 컴프리헨션으로 5줄 한 번에 읽기)
grid = [ list(map(int, input().split())) for _ in range(5) ]
# → grid[행][열] 형태 접근. 예: grid[0][1]=34, grid[1][2]=67

# 2. 최댓값 초기화
max_total = 0 # 양수 보장 시 0, 범용은 float('-inf')

# 3. 윈도우 시작점 탐색: i, j = 0~2 (5-3+1=3가지)
for i in range(3): # 시작 행: 0, 1, 2
    for j in range(3): # 시작 열: 0, 1, 2
        current_sum = 0
        # 4. 3x3 내부 합계 계산
        for r in range(i, i + 3): # 내부 행 r = i, i+1, i+2
            for c in range(j, j + 3): # 내부 열 c = j, j+1, j+2
                current_sum += grid[r][c]

        # 5. 최댓값 갱신
        if current_sum > max_total:
            max_total = current_sum

print(max_total)
리스트 컴프리헨션으로 2차원 입력 받기

[list(map(int, input().split())) for _ in range(5)]는 5줄을 한 번에 읽어 2차원 리스트로 만드는 패턴입니다.

# 풀어쓰면 동일한 코드:
grid = []
for _ in range(5):
    row = list(map(int, input().split()))
    grid.append(row)
