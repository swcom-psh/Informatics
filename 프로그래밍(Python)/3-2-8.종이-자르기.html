종이 자르기: 격자 프로그래밍

중첩 반복문을 사용하여 가로와 세로 규격에 맞는 점선 패턴을 출력합니다.

점선 그리기 프로젝트

고객이 원하는 가로(W)와 세로(H) 길이에 맞춰 큰 종이에 자르는 선을 표시해야 합니다. 컴퓨터는 위에서 아래로, 왼쪽에서 오른쪽으로 한 줄씩 출력하므로, 우리는 행(Row)과 열(Column)의 개념을 반복문으로 구현해야 합니다.

Step 1. 프로그램 기획 (요구 사항)

사용자로부터 가로와 세로 길이를 입력받아 지정된 형식의 점선을 출력합니다.

[출력 예시: 가로(W)=4, 세로(H)=3]

+---+---+---+---+
|   |   |   |   |
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+

━━ 경계선(border line): H+1개  |  ━━ 내용선(content line): H개

입력 조건

두 개의 정수 W(가로), H(세로)가 공백을 두고 입력됩니다.
예: 4 3

출력 조건

가로 W칸, 세로 H칸의 격자 모양을 기호 +, -, |를 조합하여 출력합니다.

Step 2. 프로그램 설계 (구조 분석)

격자는 크게 두 종류의 줄이 번갈아 가며 나옵니다.

1. 가로 경계선 (Border Line)

패턴: +---가 W번 반복되고 마지막에 +가 붙음.

예시 (W=4): +---+---+---+---+

파이썬: "+---" * 4 + "+" = "+---+---+---+---+"

2. 세로 칸 구분선 (Content Line)

패턴: |   (공백 3칸)이 W번 반복되고 마지막에 |가 붙음.

예시 (W=4): |   |   |   |   |

파이썬: "| " * 4 + "|" = "| | | | |"

🔴 원본 오류 수정 — 내용선 예시 공백 불일치

원본 Step 2에서 내용선 예시가 | | | | |(공백 1칸처럼 보임)으로 표기되었습니다.
실제 코드 "| " * w는 공백 3칸을 사용하므로 올바른 예시는 |   |   |   |   |입니다.
격자 출력 비주얼과 코드의 공백 수가 반드시 일치해야 학생이 혼란을 겪지 않습니다.

중첩 반복문의 구조

전체 세로 길이 H만큼 반복하면서, 그 안에서 가로 길이 W만큼 기호를 출력해야 합니다. 즉, 바깥쪽 루프는 '행(세로)', 안쪽 루프는 '열(가로)'을 담당합니다.

단, 이 코드에서는 문자열 곱셈("패턴" * W)을 활용해 안쪽 루프를 한 줄로 압축했습니다.

출력 구조 이해 — 경계선이 H+1개인 이유

H=3일 때 총 4개의 경계선과 3개의 내용선이 출력됩니다. 루프 밖의 마지막 print가 "바닥" 경계선 역할을 합니다.

루프 1회차
경계선 ①
+---+---+---+---+
루프 1회차
내용선 ①
|   |   |   |   |
루프 2회차
경계선 ②
+---+---+---+---+
루프 2회차
내용선 ②
|   |   |   |   |
루프 3회차
경계선 ③
+---+---+---+---+
루프 3회차
내용선 ③
|   |   |   |   |
루프 종료 후
마지막 경계선 ④
+---+---+---+---+

경계선 4개 = H(3) + 마지막 1개    내용선 3개 = H(3)개

Step 3. 프로그램 구현 (Source Code)

⚠️ for i → for _ 관례 수정

원본의 for i in range(h):에서 변수 i는 루프 내부에서 전혀 사용되지 않습니다. 파이썬 관례상 사용하지 않는 변수는 _(밑줄)로 표기합니다. 기능적 차이는 없지만 가독성과 코딩 스타일 측면에서 권장됩니다.

Python Solution (수정 완료)
# 1. 가로(w)와 세로(h) 입력 받기
w, h = map(int, input().split())

# 2. 전체 세로 횟수(h번)만큼 반복
# ✅ i 대신 _: 반복 횟수만 필요하고 변수값은 사용하지 않을 때의 파이썬 관례
for _ in range(h):
    # 가로 경계선: "+---"를 w번 반복하고 "+"로 마무리
    print("+---" * w + "+")
    # 세로 내용선: "| "(공백 3칸)을 w번 반복하고 "|"로 마무리
    print("| " * w + "|")

# 3. 루프 종료 후 마지막 "바닥" 경계선 출력 (h+1번째)
print("+---" * w + "+")
파이썬 꿀팁: 문자열 곱셈

파이썬에서는 "문자열" * 숫자를 통해 반복 패턴을 간결하게 만들 수 있습니다.

"+---" * 3 = "+---+---+---"
"| " * 3 = "| | | "
"A" * 0  = ""      # ⚠️ 0을 곱하면 빈 문자열!

코드 실행 추적 (Trace) — 입력: 2 2 (W=2, H=2)

코드가 실제로 어떻게 실행되는지 한 단계씩 따라가 봅시다.

단계 구분 실행 코드 출력
① 입력 w, h = map(int, "2 2".split()) w=2, h=2
② 루프 1회 — 경계선 print("+---" * 2 + "+") +---+---+
③ 루프 1회 — 내용선 print("| " * 2 + "|") |   |   |
④ 루프 2회 — 경계선 print("+---" * 2 + "+") +---+---+
⑤ 루프 2회 — 내용선 print("| " * 2 + "|") |   |   |
⑥ 루프 종료 후 마지막 print("+---" * 2 + "+") +---+---+

최종 출력: 경계선 3개(H+1=3), 내용선 2개(H=2) ✅

⚠️ 경계값 동작 — W=0 또는 H=0
  • W=0 입력: "+---" * 0 + "+" = "+", "| " * 0 + "|" = "|". 매 줄에 + 또는 | 하나씩만 출력됩니다. 수학적으로는 "0칸 격자"이므로 이 동작은 의도된 결과입니다.
  • H=0 입력: range(0)이 빈 범위 → 루프가 한 번도 실행되지 않음. 루프 밖 마지막 print만 실행되어 경계선 1줄만 출력됩니다.
