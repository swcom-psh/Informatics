Chapter 2. 프로그래밍의 기초

2-2-1. 산술 연산과 비교 연산

파이썬을 사용하여 수학적인 계산을 수행하고, 값의 크기를 비교하여 참/거짓을 판단하는 핵심 문법을 배워봅시다.

Think & Objective

"컴퓨터에 저장한 데이터는 어떻게 계산하고 처리할 수 있을까?"

  • 산술 연산자를 사용할 수 있다.
  • 비교 연산자를 사용할 수 있다.

산술 연산자

컴퓨터에 저장한 데이터는 일정한 규칙과 방법에 따라 계산하고 처리할 수 있다.

  • 연산 (Operation): 미리 정해 놓은 규칙과 방법으로 여러 단계를 거쳐 데이터를 계산하거나 처리하는 것
  • 연산자 (Operator): 연산을 표현하는 기호

연산자를 사용하면 연산 과정을 통해 결괏값을 만들 수 있다.

산술 연산자

산술 연산자를 사용하면 데이터를 계산하여 결과값을 만들 수 있다.

연산자 의미 예시 결과
+ 덧셈 5 + 2 7
- 뺄셈 5 - 2 3
* 곱셈 5 * 2 10
/ 나눗셈 5 / 2 2.5
// 몫 5 // 2 2
% 나머지 5 % 2 1
** 거듭제곱 5 ** 2 25

산술 연산 실습 예제

예제 기본 연산
70.py 
print(123+456)
Result

579

예제 입력받아 나누기
71.py 
a = input()
b = input()
a = int(a)
b = int(b)
print(a/b)
Note

/ 연산의 결과는 항상 실수형(float)입니다.

문자열에도 산술 연산자를 쓸 수 있습니다

문자열 연결

문자열도 연산자를 사용해 연결할 수 있다.

문자열 반복

  • 한 개의 단어를 입력받아 입력한 단어를 3회 반복 출력
  • 또는 단어와 반복 횟수를 입력받아 입력한 횟수만큼 반복 출력
예제 문자열 연결
72.py 
s1, s2 = input().split()
print(s1+s2)
설명

+ 연산자는 두 문자열을 하나로 이어줍니다.

예제 문자열 반복
73.py 
s = input()
print(s*3)
설명

* 연산자는 문자열을 정수만큼 반복합니다.

비교 연산자

비교 연산자는 두 값의 크기를 비교하거나 일치 여부를 판단할 때 사용하며, 결과는 언제나 참(True) 또는 거짓(False)인 불(bool) 자료형으로 나옵니다.

비교 연산 결과

  • True (참)
  • False (거짓)

이러한 값은 불(Boolean) 논릿값이다.

연산자 의미 사용 예시 및 결과 (A=10, B=5)
> 크다 (초과) 10 > 5 → True
< 작다 (미만) 10 < 5 → False
>= 크거나 같다 (이상) 10 >= 5 → True
<= 작거나 같다 (이하) 10 <= 5 → False
== 같다 10 == 5 → False
!= 같지 않다 (다르다) 10 != 5 → True

비교 연산 실습 예제

예제 크기 비교
75.py 
print(123<456)
Result

True

예제 일치 비교
76.py 
a = input()
b = input()
a = int(a)
b = int(b)
print(a==b)
설명

변수 a와 b에 저장된 값이 같으면 True, 다르면 False를 출력합니다.

예제 실수 크기 비교
77.py 
a = input()
b = input()
a = float(a)
b = float(b)
print(a<=b)
Note

실수(float) 데이터도 비교 연산이 가능합니다.

예제 문자열 일치 비교
78.py 
s1, s2 = input().split()
print(s1==s2)
설명

문자열의 내용이 완전히 똑같은지 확인합니다.

학습 정리

나머지 연산의 활용

나머지 연산(%)은 짝수/홀수 판별이나 배수를 확인할 때 매우 중요하게 쓰입니다.

주의사항

같다는 기호는 == (두 개)입니다. = (한 개)는 변수에 값을 대입할 때만 씁니다.
