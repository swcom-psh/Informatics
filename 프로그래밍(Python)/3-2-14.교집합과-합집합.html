교집합과 합집합 계산기

집합(Set) 자료구조를 활용하여 두 집합의 공통 원소와 전체 원소를 오름차순으로 추출합니다.

집합 자료구조로 수학 문제 풀기

수학 시간에 배운 교집합·합집합을 파이썬으로 단 두 줄에 구현할 수 있습니다. 파이썬의 set 자료구조는 수학의 집합과 동일한 성질(중복 없음, 순서 없음)을 가지며, &, | 연산자로 집합 연산을 직관적으로 표현합니다.

Step 1. 프로그램 기획 (문제 정의)

두 개의 숫자 집합을 입력받아 수학의 집합 연산을 수행합니다. 결과는 반드시 오름차순으로 정렬하여 출력해야 합니다.

핵심 개념: 집합(Set) 자료구조

중복을 허용하지 않으며 순서가 없는 데이터 묶음입니다.

원소별 소속 관계:

1 (A만) 2 (A만) 3 (A∩B) 4 (A∩B) 5 (B만) 6 (B만)
⚠️ 입력 형식 명확화 — 원본에 누락

원본 코드는 set(map(int, input().split()))으로 읽지만 입력 형식이 전혀 설명되지 않습니다. 아래 형식을 확인하세요.

# 입력 방식: 각 줄에 공백으로 구분된 정수들
# (원소 개수를 먼저 입력받는 문제도 있으니 문제 조건 확인 필수!)

# 형식 예시 ①: 원소를 공백으로 한 줄에 입력
1 2 3 4 # → set_a = {1, 2, 3, 4}
3 4 5 6 # → set_b = {3, 4, 5, 6}

교집합 (Intersection)

두 집합에 공통으로 포함된 원소들을 추출합니다.
파이썬 연산자: &

{1,2,3,4} & {3,4,5,6} = {3,4}

합집합 (Union)

두 집합의 모든 원소를 합칩니다. (중복 자동 제거)
파이썬 연산자: |

{1,2,3,4} | {3,4,5,6} = {1,2,3,4,5,6}

Step 2. 프로그램 설계 (알고리즘)

입력된 데이터를 집합으로 변환하고 정렬하는 과정을 설계합니다.

알고리즘 순서 설계
  1. 두 집합의 원소 데이터를 각각 입력받음
  2. 입력받은 데이터를 set()으로 변환
  3. 집합 연산(&, |) 수행
  4. 결과를 sorted()로 오름차순 정렬 (리스트 변환 포함)
  5. 교집합이 비어있으면(공집합) 0 출력, 아니면 원소 출력

집합 연산자 & 메서드 비교

연산 연산자 메서드 결과 (예시)
교집합 set_a & set_b set_a.intersection(set_b) {3, 4}
합집합 set_a | set_b set_a.union(set_b) {1, 2, 3, 4, 5, 6}
차집합 set_a - set_b set_a.difference(set_b) {1, 2}
대칭차 set_a ^ set_b set_a.symmetric_difference(set_b) {1, 2, 5, 6}

핵심 문법 2가지

# 1. sorted() — set을 직접 정렬 (list() 불필요)
s = {3, 1, 4, 1, 5} # set은 중복 제거, 순서 없음
sorted(s) # [1, 3, 4, 5] — sorted()는 iterable 직접 수용
sorted(list(s)) # 동일 결과 — list() 변환 불필요 (중복 코드)
# 2. print(*리스트) — 언패킹으로 공백 구분 출력
result = [1, 3, 4]
print(result) # [1, 3, 4] ← 대괄호 포함 출력
print(*result) # 1 3 4 ← 공백 구분, 대괄호 없음 ✅

Step 3. 프로그램 구현 (Python)

집합 자료구조와 내장 함수를 활용한 구현 예시입니다.

💡 코드 개선 포인트 — list() 제거

원본의 sorted(list(set_a & set_b))에서 list()는 불필요합니다. sorted()는 set을 포함한 모든 iterable을 직접 받습니다. 아래 완성 코드에서 간소화했습니다.

⚠️ print(0) — 공집합 처리 이유 설명 필요

원본 코드의 print(0)은 교집합이 공집합일 때 숫자 0을 출력합니다. 이는 문제 조건에서 "공집합이면 0을 출력하라"는 요구사항이 있을 때만 올바른 처리입니다. 문제 조건을 반드시 확인하세요. 원소가 없음을 나타내는 빈 줄 출력이 요구될 수도 있습니다.

Python Solution
# 1. 두 집합 입력 (공백 구분 정수, 한 줄씩)
set_a = set(map(int, input().split()))
set_b = set(map(int, input().split()))

# 2. 교집합 연산 및 오름차순 정렬
# sorted()는 set 직접 수용 → list() 변환 불필요
inter = sorted(set_a & set_b)
if not inter: # 공집합: not [] == True
    print(0) # 문제 조건: 공집합이면 0 출력
else:
    print(*inter) # *언패킹: [3,4] → "3 4" (공백 구분)

# 3. 합집합 연산 및 오름차순 정렬
uni = sorted(set_a | set_b)
print(*uni) # 합집합은 항상 원소 있음 (A, B 모두 비어있지 않으면)
주의사항

집합(set)은 자체적으로 순서가 없으므로, 출력 시 반드시 sorted()를 사용하여 오름차순 정렬 과정을 거쳐야 채점 시스템에서 정답으로 인정됩니다. 또한 집합의 순서는 파이썬 버전·실행 환경에 따라 달라지므로 절대 순서에 의존하지 마세요.

코드 실행 추적 (Trace)

입력: 1 2 3 4 / 3 4 5 6

단계 코드 계산 과정 결과
① set_a = set(map(int, ...)) "1 2 3 4" → int 변환 → set set_a = {1, 2, 3, 4}
② set_b = set(map(int, ...)) "3 4 5 6" → int 변환 → set set_b = {3, 4, 5, 6}
③ set_a & set_b 공통 원소 추출 {3, 4} (순서 미보장)
④ inter = sorted({3, 4}) 오름차순 정렬 [3, 4] ✅
⑤ if not inter → False [3, 4]는 비어있지 않음 else 분기 실행
⑥ print(*inter) *[3, 4] 언패킹 출력: 3 4
⑦ set_a | set_b 모든 원소 합집합 {1, 2, 3, 4, 5, 6} (순서 미보장)
⑧ uni = sorted(...)
print(*uni)		 정렬 후 언패킹 출력 출력: 1 2 3 4 5 6
