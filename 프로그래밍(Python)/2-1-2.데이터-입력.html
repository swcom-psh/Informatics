Chapter 2. 프로그래밍의 기초

2-1-2. 데이터 입력 및 변수 활용

사용자로부터 값을 입력받아 변수에 저장하고, 저장된 데이터를 자유자재로 다루는 방법을 마스터해 봅시다.

Think & Objective

"여러 가지 데이터를 컴퓨터로 다루기 위해서는 어떻게 입력하고 출력해야 할까?"

  • 변수를 사용해 데이터를 저장할 수 있다.
  • 원하는 형식으로 데이터를 출력할 수 있다.

0. 시작하기 전에: 컴퓨터에게 내 생각 전달하기

앞서 배운 print()가 컴퓨터가 우리에게 말을 거는 '확성기'였다면, 이번엔 우리가 컴퓨터에게 대답을 해줄 차례입니다.

파이썬에서 input()은 "사용자가 키보드로 무언가를 입력할 때까지 가만히 기다려!"라는 뜻을 가진 마법의 명령어입니다. 우리가 타자를 치고 엔터(Enter) 키를 누르는 순간, 컴퓨터가 그 글자를 쏙 가져가서 기억하게 된답니다.

지금은 "아, 내가 키보드로 친 글자를 컴퓨터 안으로 집어넣는 거구나!" 정도로만 가볍게 생각하고 출발해 보세요. 이 데이터를 어떻게 숫자로 바꾸고 여러 개로 쪼개는지 아래에서 아주 쉽고 완벽하게 다룰 거예요!

변수(Variable)의 기본

변수란?

변수란 데이터가 저장된 위치를 가리키는 특별한 이름(메모리 공간)입니다. 여러 가지 데이터를 저장한 후 이를 불러 사용할 때 변수(variable)를 만들어 사용할 수 있습니다.

변수를 활용하면 여러 가지 이름으로 데이터를 저장할 수 있고, 필요할 때마다 편리하게 불러 재사용이 가능합니다.

  • 할당 연산자(=): 값을 변수에 저장할 때 사용합니다.
  • 동적 타이핑: 파이썬은 변수에 들어있는 값에 따라 자료형이 결정됩니다.

▲ 변수 이름으로 사용할 수 없는 것 (주의!)

프로그래밍 언어에서 이미 문법적인 용도로 사용 중인 예약어(if, for, while 등)나 내장 함수 이름(print, input 등)은 변수 이름으로 사용할 수 없습니다.

다양한 실습 예제

예제 변수 선언과 여러 번 출력하기
53.py 
s = "Hello"
print(s)
print(s)
print(s)
예제 변수 값 덮어쓰기
54.py 
n = 0
print(n)
n = 1
print(n)
print(n)
Tip

프로그램 실행 도중 새로운 값을 변수에 저장하면 새로운 값으로 덮어쓰기(overwrite) 되고, 다른 값으로 바꾸기 전까지 그대로 유지됩니다.

예제 변수 여러 개 활용하여 출력 순서 바꾸기
55.py 
s1 = "Hello"
s2 = "Python"
s3 = "World"
print(s1, s2, s3)
print(s2, s3, s1)
print(s3, s1, s2)
Tip

여러 개의 값을 저장했다가 사용해야 할 경우, 위와 같이 필요한 개수만큼 여러 개의 변수를 만들어 사용 가능합니다.

리스트(List)의 기초

리스트(List)란?

여러 개의 값을 하나의 리스트로 만들어 변수에 저장하였다가 사용해야 할 경우 대괄호 [ ] 로 데이터를 나열해 리스트 변수로 저장합니다.

📌 리스트 요소 접근 방법

  • 리스트 변수에 저장된 값을 사용할 경우 대괄호 [ ] + 참조 번호를 함께 사용합니다.
  • 중요: 리스트의 첫 번째 데이터 번호는 항상 0입니다. (1이 아닙니다!)
예제 리스트 인덱싱
56_2.py 
d = [1, 2, 3, 4, 5]
print(d[4], d[3], d[2], d[1], d[0])
Result

5 4 3 2 1

input() 함수와 데이터 입력

input() 함수란?

input() 함수를 사용하면 키보드를 통해 입력한 문자열을 변수에 바로 저장할 수 있습니다. 프로그램은 사용자가 값을 입력할 때까지 일시 정지합니다.

예제 한 줄 입력받아 출력하기
58.py 
s = input()
print(s)
예제 여러 줄 연속으로 입력받기
59.py 
s1 = input()
s2 = input()
print(s1)
print(s2)
Caution

input()의 결과는 항상 '문자열'입니다.

숫자로 계산하고 싶다면 나중에 배울 int() 변환이 필요합니다.

잠깐! 문자열을 숫자로 어떻게 바꿀까요?

앞서 input()으로 받은 값은 모두 문자열(String)이라고 배웠습니다. 만약 "10"과 "20"을 입력받아 더하면 수학적 계산인 30이 되는 것이 아니라, 글자가 연결되어 "1020"이 되어버립니다. 이를 방지하기 위해 int() 함수를 사용하여 정수(Integer)로 변환해주어야 합니다.

기초 예제 문자를 숫자로 변환하여 계산하기
supplement_1.py 
# 글자 "10"을 입력받아 a에 저장합니다.
a = input()
# a에 들어있는 글자 "10"을 진짜 숫자 10으로 바꿉니다.
a = int(a)

b = input()
b = int(b)

# 이제 정상적으로 수학 덧셈이 됩니다!
print(a + b)

효율적인 입력 처리 기법

자료형 변환의 중요성

프로그래밍 언어에서는 데이터의 종류에 따라 정확한 자료형을 사용해야 합니다. 문자열 그대로는 수학적 계산이 불가능하므로, 숫자로 사용하려면 알맞은 형태(int, float 등)로 변환하는 과정이 꼭 필요합니다.

예제 split()의 결과 통째로 확인해보기
60_sup.py 
# "사과 바나나" 라고 입력해 보세요.
s = input().split()
print(s)  # 결과: ['사과', '바나나'] 처럼 대괄호로 묶여서 나옵니다.

이렇게 대괄호 []로 묶인 형태를 리스트(List)라고 부릅니다. 이 리스트에서 원하는 것만 쏙쏙 골라내고 싶을 때 아래의 예제 61번처럼 [번호]를 사용합니다.

예제 공백으로 분리하기 (여러 변수에 저장)
60.py 
s1, s2 = input().split()
print(s1)
print(s2)
흔히 하는 실수 (에러 주의!)

입력하는 단어 개수와 변수 개수가 다르면 에러가 납니다!

위 예제에서 변수는 s1, s2 두 개인데, 실수로 단어를 3개 이상 입력하면 ValueError (too many values to unpack)라는 무서운 에러가 발생하고 프로그램이 멈춥니다. 반대로 두 개를 입력해야 하는데 사용자가 숫자 한 개만 입력해도 에러가 발생합니다. 반드시 변수의 개수만큼만 공백으로 띄워서 입력해야 합니다.

예제 공백으로 분리하기 (리스트 인덱싱)
61.py 
s = input().split()
print(s[0])
print(s[1])
print(s[2])
예제 특정 구분자 활용 (예: 시각 분리)
63.py 
h, m, s = input().split(':')
print(h)
print(m)
print(s)
Tip

split(':')와 같이 괄호 안에 문자를 넣으면 해당 문자를 기준으로 데이터를 자릅니다.

map 함수의 활용

마법의 주문: map() 함수 파헤치기

split()을 사용해 데이터를 나누면, ['1', '2', '3'] 처럼 여러 개의 '문자열'이 담긴 리스트가 만들어집니다. 이때 리스트 안의 모든 문자를 한 번에 숫자로 바꾸려면 매우 번거롭겠죠?

이때 구원투수처럼 등장하는 것이 바로 map(적용할 함수, 데이터)입니다. map(int, 입력데이터)라고 작성하면, 잘려진 여러 개의 문자 데이터를 순식간에 모두 정수(int)로 싹 바꿔줍니다.

1단계: 입력

"1 2 3"

2단계: split()

['1', '2', '3']

▶
3단계: map(int, ...)

[1, 2, 3]

흐름 한 줄 코드의 실제 동작 단계

list(map(int, input().split()))
이 한 줄은 사실 아래 4단계를 순서대로 실행한 것과 같습니다.

1
a = input()

사용자의 입력을 한 줄의 통문자열로 받아옵니다. "1 2 3 4 5"

2
b = a.split()

공백을 기준으로 쪼개어 문자열 리스트를 만듭니다. ['1', '2', '3', '4', '5']

3
c = map(int, b) WAITING

각 요소에 int()를 적용할 준비만 마친 상태입니다.

Lazy Evaluation: 아직 변환 전
4
d = list(c)

실제로 함수를 실행하여 정수형 리스트를 최종 생성합니다. [1, 2, 3, 4, 5]

풀버전 62_step.py 
a = input()        # 문자열 입력
b = a.split()      # 문자열 리스트로 분리
c = map(int, b)    # 정수로 변환 준비
d = list(c)        # 정수 리스트 완성
print(d)
한 줄 압축 62.py 
d = list(map(int, input().split()))
print(d)
Point

풀버전의 4줄을 1줄로 압축한 코드입니다. 동작은 완전히 동일합니다.

예제 숫자 리스트로 즉시 변환 (map)
62.py 
d = list(map(int, input().split()))
print(d[0], d[1], d[2], d[3], d[4])
예제 콤마(,) 구분자를 숫자 리스트로 변환
64.py 
d = list(map(int, input().split(',')))
print(d[0], d[1], d[2], d[3], d[4], sep='\n')
Advanced

map(int, ...)는 모든 입력값을 한꺼번에 정수로 바꾸는 마법 같은 코드입니다!

심화 학습 변수의 종류 (Scope) 및 전역 변수 활용

변수의 종류 (Scope)

미리 알아두기: 함수(def)란?

아래 예제 57번 코드에서 def f(n): 이라는 낯선 문법이 등장합니다. 이는 '나만의 명령어(함수)를 만드는 방법'입니다. 함수 안에서 바깥에 있는 변수(전역 변수)의 값을 마음대로 수정하려면 global 변수이름 이라고 명시해주어야 한다는 점에만 집중해서 코드를 읽어보세요!

전역 변수 (global variable)

프로그램의 모든 위치에서 참조해 사용할 수 있는 변수입니다.

지역 변수 (local variable)

함수 안과 같이 특정 범위 내에서만 참조해 사용할 수 있는 변수입니다.

매개 변수 (parameter)

함수 실행에 필요한 값을 함수 안으로 전달하기 위해 사용하는 변수입니다.

예제 전역 변수(Global) 활용
57.py 
s = 0
def f(n):
  global s
  k = 0
  while k < n:
    k = k + 1
    s = s + k
f(3)
print(s)
