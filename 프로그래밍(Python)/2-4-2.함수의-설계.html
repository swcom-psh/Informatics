Chapter 2. 프로그래밍의 기초

2-4-2. 함수의 설계

파이썬 내장 함수 외에, 나만의 기능을 수행하는 함수를 직접 정의하고 호출하는 방법과 변수의 유효 범위, 외장 모듈 응용까지 학습합니다.

Think & Objective

"함수는 어떻게 만들어졌으며, 내가 원하는 작업을 수행하는 함수는 어떻게 만들 수 있을까?"

  • 사용자 정의 함수를 설계하여 사용할 수 있다.
  • 함수의 반환값과 매개 변수를 적절하게 고려하여 코드를 작성할 수 있다.

함수의 정의와 호출

파이썬에서 함수를 새로 만들 때는 def 키워드를 사용합니다. 함수를 정의만 한다고 해서 바로 실행되는 것은 아니며, 반드시 함수의 이름을 불러서 호출해야 내부의 코드가 동작합니다.

함수 정의 실습 예제

예제 함수만 정의했을 때
132.py 
def f():
  print("Hello")
설명

함수 f()를 만들기만 하고 호출하지 않았으므로, 화면에는 아무것도 출력되지 않습니다.

예제 함수 호출하기
134.py 
def f():
  print("Hello")

f()
print("World")
Result

Hello
World

설명

f()를 호출하면 정의된 내부로 들어가 Hello를 출력한 뒤, 원래 자리로 돌아와 World를 출력합니다.

return 문의 역할

return 명령어는 두 가지 중요한 역할을 수행합니다.
1. 즉시 함수를 종료하고 호출했던 원래 위치로 돌아갑니다.
2. 함수가 작업한 결과값을 호출한 곳으로 반환(전달)합니다.

return 실습 예제

예제 함수의 조기 종료
135.py 
def f():
  print("Hello")
  return
  print("World")

f()
Result

Hello

설명

Hello 출력 후 return을 만나 함수가 즉시 끝나버리므로, 그 아래의 World는 영원히 출력되지 않습니다.

예제 결과값 반환하기
136.py 
def f():
  return 3

print(f())
Result

3

설명

f() 함수가 호출된 자리가 return 뒤에 적힌 값인 3으로 치환됩니다. 따라서 print(3)이 수행됩니다.

매개변수 (Parameters)

함수를 호출할 때 외부에서 값을 넘겨주고 싶다면 매개변수를 사용합니다.
괄호 안에 변수 이름을 적어두면, 함수를 호출할 때 전달한 값(인자)이 그 변수에 순서대로 쏙쏙 저장되어 함수 내부에서 활용할 수 있습니다.

매개변수 실습 예제

예제 단일 매개변수
137.py 
def f(n):
  print(n)

f(3)
Result

3

설명

f(3)으로 호출하면 매개변수 n에 3이 들어가고, 내부에서 print(3)이 실행됩니다.

예제 복수 매개변수와 연산
138.py 
def f(a, b):
  return a+b

print(f(3, 5))
Result

8

설명

3은 a에, 5는 b에 전달됩니다. 두 수의 합인 8이 반환되어 최종적으로 화면에 출력됩니다.

함수와 제어 구조의 활용

함수 내부에서는 이전에 배웠던 if-else문이나 while, for 등의 제어/반복 구조를 완벽하게 똑같이 사용할 수 있습니다. 이를 통해 복잡한 로직을 함수 하나로 묶어 깔끔하게 관리합니다.

응용 실습 예제

예제 함수 내에 if-else 사용
139.py 
def f(a, b):
  if a<b:
    return a
  else:
    return b

print(f(3, 5))
Result

3

설명

전달받은 두 수 중 더 작은 수를 판단하여 반환하는 함수입니다.

예제 함수 내에 while 반복문 사용
140.py 
def f(n):
  k = 0
  while k<n:
    k = k+1
    print(k)

f(3)
Result

1
2
3

설명

매개변수 n 만큼 1부터 숫자를 순서대로 출력하는 기능을 수행합니다.

지역 변수와 전역 변수 (Scope)

함수 안에서 만들어진 변수는 지역 변수(Local)로 함수가 끝나면 사라집니다.
반면, 함수 밖에서 만들어진 전역 변수(Global)를 함수 내부에서 직접 수정하려면 반드시 global 키워드를 명시해야 합니다.

변수 스코프와 누적 합 실습

예제 지역 변수를 사용해 합 구하기
141.py 
def f(n):
  k = 0
  s = 0
  while k<n:
    k = k+1
    s = s+k
  return s

print(f(3))
Result

6

설명

함수 안에서 선언된 s는 지역 변수입니다. 1부터 3까지의 합(1+2+3=6)을 계산한 후, return s를 통해 그 값을 밖으로 반환합니다.

예제 전역 변수(global) 활용
142.py 
s = 0
def f(n):
  global s
  k = 0
  while k<n:
    k = k+1
    s = s+k

f(3)
print(s)
Result

6

설명

global s를 선언하면, 함수 밖(맨 위)에 있는 전역 변수 s를 함수 내부에서 직접 수정합니다. 결과적으로 return 없이도 밖의 변수 값이 변합니다.

예제 for문을 활용한 함수 설계
143.py 
def f(n):
  s = 0
  for k in range(1, n+1):
    s = s+k
  return s

n = int(input())

print(f(n))
설명

while문 대신 for문을 사용하면 누적 합 연산을 훨씬 간결하게 구현할 수 있습니다. 사용자로부터 입력받은 n을 매개변수로 전달하여 그 합을 구합니다.

외장 모듈 활용 심화 (random)

미리 만들어진 강력한 도구인 외장 모듈 중에서도 무작위 값을 생성하는 random 모듈을 실습해 봅니다. 게임이나 시뮬레이션 프로그램 등을 만들 때 필수적으로 사용됩니다.

random 모듈 실습 예제

예제 난수 생성기 (randint)
146.py 
from random import *
n = int(input())
for i in range(n):
  print(randint(1, 100))
설명

입력받은 n번 만큼, 1부터 100 사이의 무작위 정수(난수)를 반환하는 randint(1, 100) 함수를 반복 실행합니다.

예제 난수 고정 (seed)
147.py 
from random import *
seed(1)
n = int(input())
for i in range(n):
  print(randint(1, 100))
설명

원래 난수는 실행할 때마다 값이 달라지지만, seed(1) 처럼 기준값을 고정해두면 매번 같은 순서와 값의 난수가 발생하여 디버깅이나 테스트에 유리합니다.

학습 정리

함수의 기본 뼈대

def 함수명(매개변수): 로 만들고 함수명(값) 으로 실행시킵니다. 호출하지 않은 함수는 단지 도구상자에 들어있는 도구와 같습니다.

데이터와 변수 스코프

함수에 값을 넣을 땐 매개변수, 밖으로 뺄 땐 return을 사용합니다. 함수 외부의 변수를 수정하고 싶다면 global 선언이 필요합니다.
