선택 구조

반복 구조

Section 1

기초 다지기: C언어에서 '참'과 '거짓'의 비밀

본격적으로 조건문을 다루기 전에 꼭 알아야 할 중요한 규칙이 있습니다. C언어는 참(True)과 거짓(False)을 오직 '숫자'로만 구분합니다. 조건식의 결과가 0이면 거짓(False)으로 판단하고, 0이 아닌 모든 수(1, -1, 13 등)는 참(True)으로 판단하여 코드를 실행합니다.

꼭 알아야 할 비교 연산자

  • A == B A와 B는 같다 (=는 대입, ==가 비교!)
  • A != B A와 B는 다르다
  • A > B A가 B보다 크다
  • A < B A가 B보다 작다
  • A >= B A가 B보다 크거나 같다
  • A <= B A가 B보다 작거나 같다

💡 독학 포인트

다음 예제에서 if (1)이나 if (-1) 같은 코드를 보게 될 것입니다. 당황하지 마세요!

  • if (1) : 0이 아니므로 무조건 참 (실행됨)
  • if (-1) : 0이 아니므로 무조건 참 (실행됨)
  • if (0) : 0이므로 무조건 거짓 (실행 안 됨)

"C언어에서는 0만 아니면 다 참이다!"라고 기억해 두세요.

Section 2

기초 다지기: 조건을 하나로 묶기 — 논리 연산자

비교 연산자만으로는 복잡한 상황을 표현하기 어렵습니다. 예를 들어, "키가 130cm 이상이고, 나이가 10살 이상인가?"를 확인하려면 두 개의 조건을 동시에 검사해야 합니다. 이때 사용하는 것이 논리 연산자입니다.

&& AND (그리고)

양쪽 조건이 모두 참이어야 전체가 참이 됩니다.

if (age >= 10 && height >= 130)

|| OR (또는)

양쪽 조건 중 하나라도 참이면 전체가 참이 됩니다.

if (money >= 1000 || hasCard == 1)

! NOT (부정)

참을 거짓으로, 거짓을 참으로 반대로 뒤집습니다.

if (!isRainy) // 비가 안 오면

수학처럼 쓰면 안 돼요!

수학에서는 10 <= age <= 20 이라고 쓰지만, C언어에서는 이렇게 쓰면 컴퓨터가 오작동합니다. 반드시 논리 연산자를 사용하여 age >= 10 && age <= 20 처럼 두 조건으로 나누어 묶어주어야 합니다.

Section 4

선택 구조 — 단순 if 문

단순 if 문은 조건식이 참일 때만 특정 코드를 실행하는 구조입니다. 조건이 거짓이면 if 블록 안의 코드는 완전히 생략되고, 다음 코드로 넘어갑니다.

if 문법 구조 

if (조건식) // 조건식이 참(0이 아닌 값)이면 아래 블록을 실행
{
    실행할 코드; // 조건이 참일 때만 실행
}
// 조건이 거짓이면 if 블록을 건너뛰고 여기서부터 계속 실행
  • ● 소괄호 ( ) 안의 조건식이 참(0이 아닌 값)이면, 중괄호 { } 안의 코드를 실행합니다.
  • ● 조건이 거짓(0)이면 중괄호 안의 코드는 완전히 건너뜁니다.

if 실행 흐름

조건이 참이면 if 블록 실행, 거짓이면 아무 것도 실행하지 않고 바로 다음 코드로 이동합니다.

학생들이 가장 많이 하는 실수 BEST 1

수학에서의 '같다'는 =이지만, C언어 조건문에서 '같다'는 반드시 ==를 사용해야 합니다. =는 대입 연산자입니다.

오류: if (score = 100) { ... } // score에 100을 대입 → 조건식 결과가 100(참)이 되어 항상 실행됨
정답: if (score == 100) { ... } // score가 100인지 비교

코딩 팁: 중괄호 { }는 언제 생략할 수 있을까?

if문 아래에 실행할 코드가 딱 1줄뿐이라면 중괄호를 생략할 수 있습니다. (바로 아래 98.c 예제처럼요!)
하지만 코드가 2줄 이상인데 중괄호를 생략하면, 첫 번째 줄만 if문에 포함되고 나머지는 조건과 상관없이 무조건 실행되는 치명적인 버그가 발생합니다.
초보자라면 코드가 1줄이더라도 항상 중괄호를 쓰는 습관을 들이는 것이 좋습니다.

예제 단순 if문 — 참/거짓 판별 (98.c)
98.c 
#include <stdio.h>
int main()
{
    int i = 13, j = 5;
    if (-1) putchar('A');   /* -1은 0이 아니므로 참 → 'A' 출력 */
    if (1)  putchar('B');   /* 1은 0이 아니므로 참  → 'B' 출력 */
    if (13) putchar('C');   /* 13은 0이 아니므로 참 → 'C' 출력 */
    if (0)  putchar('D');   /* 0이므로 거짓          → 건너뜀  */
    if (i > j)              /* 13 > 5 이므로 참      → 블록 실행 */
    {
        putchar('E');
        putchar('F');
    }
    if (i < j)              /* 13 < 5 이므로 거짓    → 건너뜀  */
        putchar('G');
    putchar('H');           /* if와 무관, 무조건 실행 → 'H' 출력 */
    return 0;
}
단계별 실행 추적
코드 조건 판단 실행 여부 출력
if (-1) -1 ≠ 0 → 참 ✅ 실행 A
if (1) 1 ≠ 0 → 참 ✅ 실행 B
if (13) 13 ≠ 0 → 참 ✅ 실행 C
if (0) 0 = 0 → 거짓 ⛔ 건너뜀 —
if (i > j) → if (13 > 5) 13 > 5 → 참 ✅ 실행 E, F
if (i < j) → if (13 < 5) 13 < 5 → 거짓 ⛔ 건너뜀 —
putchar('H') if문 아님 — 무조건 실행 ✅ 실행 H
최종 출력 결과
ABCEFH

Section 5

선택 구조 — if-else 문

단순 if문은 조건이 참일 때만 코드를 실행합니다. 반면 if-else 문은 조건이 참일 때 실행할 코드와 거짓일 때 실행할 코드를 각각 따로 지정할 수 있습니다.

if-else 문법 구조 

if (조건식) // 조건이 참이면 ↓
{
    참일 때 실행할 코드;
}
else // 조건이 거짓이면 ↓
{
    거짓일 때 실행할 코드;
}
  • 1.if (조건식): 조건이 참이면 첫 번째 중괄호 { } 안의 코드를 실행합니다.
  • 2.else: 조건이 거짓이면 else 뒤의 중괄호 { } 안의 코드를 실행합니다.
  • 3.둘 중 하나만 실행됩니다. if 블록과 else 블록은 절대 동시에 실행되지 않습니다.

if-else 실행 흐름

if 블록과 else 블록 중 딱 하나만 실행되고, 이후 프로그램은 동일하게 계속됩니다.

예제 if-else 문 — 합격/불합격 판별 (99.c)
99.c 
#include <stdio.h>
int main()
{
    int standard = 80, score = 76;

    /* standard(80) < score(76)  →  80 < 76  →  거짓 */
    if (standard < score)
    {
        putchar('P');   /* 합격(Pass): 조건이 참일 때 */
    }
    else
    {
        putchar('F');   /* 불합격(Fail): 조건이 거짓일 때 */
    }
    return 0;
}
실행 결과
F // standard(80) < score(76) → 80 < 76 → 거짓 → else 블록 실행 → 'F' 출력

Section 6

선택 구조 — if-else if 문 (다중 분기)

비교해야 할 조건이 셋 이상일 때는 else if를 연결해서 여러 경우를 순서대로 검사합니다. 위에서부터 조건을 차례로 확인하다가, 처음으로 참인 조건의 블록을 실행하고 나머지는 모두 건너뜁니다.

if-else if 문법 구조 

if (조건 A) // 조건 A를 먼저 검사
{
    A가 참일 때 실행;
}
else if (조건 B) // A가 거짓이면 B를 검사
{
    B가 참일 때 실행;
}
else // A도 B도 거짓이면
{
    나머지 경우 실행;
}
  • 1.if (조건 A): 가장 먼저 검사합니다. 참이면 바로 실행하고, 아래 else if / else는 모두 건너뜁니다.
  • 2.else if (조건 B): A가 거짓일 때만 B를 검사합니다. 참이면 실행하고 나머지는 건너뜁니다.
  • 3.else: 앞의 모든 조건이 거짓일 때 실행됩니다. 생략 가능합니다.

다중 조건문(if-else if-else) 상세 순서도

계단식 검사: 위에서부터 하나씩 검사하며 내려옵니다.
else 블록 추가: 모든 조건이 거짓일 때 실행되는 'else 실행' 블록을 추가했습니다.
즉시 탈출: 어떤 문장이든 실행이 완료되면 파란색 메인 경로를 타고 전체 조건문을 즉시 빠져나갑니다.

예제 if-else if 문 — 키 비교 (100.c)
100.c 
#include <stdio.h>
int main()
{
    int standard = 173, height = 168;

    /* 조건 A: standard(173) > height(168) → 173 > 168 → 참! */
    if (standard > height)
    {
        printf("평균 이하");      /* 조건 A가 참이므로 이것만 실행 */
    }
    else if (standard == height)  /* 조건 A가 참이므로 여기는 검사 안 함 */
    {
        printf("평균");
    }
    else                           /* 조건 A가 참이므로 여기도 건너뜀 */
    {
        printf("평균 이상");
    }
    return 0;
}
실행 결과
평균 이하 // standard(173) > height(168) → 참 → 첫 번째 if 블록만 실행, 나머지 건너뜀

Section 7

선택 구조 — 중첩 if 문 (Nested if)

else if가 여러 조건을 "순서대로" 검사하는 구조라면, 중첩 if문은 조건 안에 또 다른 조건을 넣어 "단계적으로" 검사하는 구조입니다.

중첩 if 문 구조 

if (조건식 A) // 조건 A가 참이면 ↓
{
    if (조건식 B) // 조건 B가 참이면 ↓
    {
        A가 참이고 B도 참일 때 실행;
    }
    else
    {
        A는 참이면서 B는 거짓일 때 실행;
    }
}
else // 조건 A가 거짓이면 ↓
{
    A가 거짓일 때 실행(B와는 관계 없음);
}
  • 1. if (조건식 A): 조건 A가 참이면 첫 번째 블록 안 코드가 실행되고, 그 안에서 조건 B를 검사합니다.
  • 2. if (조건식 B) / else: 조건 A가 참일 때, B가 참이면 B 블록 실행, 거짓이면 else 블록 실행.
  • 3. else: 조건 A가 거짓이면 A 블록 안의 중첩 if는 무시되고 else 블록 실행.
  • 4. 실행 원칙: 항상 하나의 경로만 실행되며, 동시에 여러 블록이 실행되지 않습니다.
💡 예를 들어, "필기시험 합격자 중에서, 실기시험까지 합격한 사람"을 찾으려면 두 단계가 필요합니다. 1단계 조건(필기 합격)을 통과한 사람에게만 2단계 조건(실기 합격)을 적용할 때 중첩 if를 사용합니다.
else if — 순서대로 비교

조건 A, B, C 중 하나만 선택. 조건들은 서로 독립적입니다.

if (score >= 90) ... A학점
else if (score >= 80) ... B학점
else ... C학점
중첩 if — 단계적으로 좁혀가기

조건 A를 통과한 경우에만 조건 B를 검사합니다.

if (필기 합격) {
    if (실기 합격) ... 최종 합격
    else ... 실기 불합격
} else ... 필기 불합격
예제 중첩 if문 — 운전면허 최종 합격자 판별
nested_if.c 
#include <stdio.h>

int main() {
    int written_score = 85;    /* 필기 점수 */
    int practical_score = 70;  /* 실기 점수 */

    /* 1단계: 필기 점수가 80점 이상인가? */
    if (written_score >= 80) {
        printf("1차 필기 합격!\n");

        /* 2단계(중첩): 실기 점수도 60점 이상인가? */
        /* → 1단계를 통과한 사람에게만 이 조건을 검사합니다 */
        if (practical_score >= 60) {
            printf("최종 운전면허 취득을 축하합니다!\n");
        } else {
            printf("아쉽지만 실기에서 불합격하셨습니다.\n");
        }

    } else {
        printf("필기 불합격입니다.\n");
        /* 필기를 통과하지 못하면 실기 조건은 검사하지 않습니다 */
    }

    return 0;
}
실행 결과
1차 필기 합격! 최종 운전면허 취득을 축하합니다! // written_score(85) >= 80 → 참 → 내부 if 검사 // practical_score(70) >= 60 → 참 → 최종 합격 출력

Section 8

선택 구조 — switch-case 문

비교해야 할 경우의 수가 많을 때, if-else if를 길게 나열하는 대신 switch문을 사용하면 코드가 훨씬 깔끔해집니다. 단, switch는 정수(int) 또는 문자(char) 값의 정확한 일치만 검사할 수 있습니다. 범위 비교(예: score >= 80)는 if문을 사용해야 합니다.

switch-case 문법 구조 

switch (변수 또는 식) // 이 값을 아래 case들과 비교
{
    case 값1: // 변수 == 값1 이면 여기서 시작
        실행할 코드;
        break; // switch 전체를 빠져나감 (필수!)

    case 값2:
        실행할 코드;
        break;

    default: // 어떤 case와도 일치하지 않으면 실행
        실행할 코드;
}
  • 1.switch (변수): 소괄호 안의 값을 각 case 값과 차례대로 비교합니다.
  • 2.case 값: 변수의 값이 이 숫자(또는 문자)와 정확히 일치하면 이 위치에서 실행을 시작합니다.
  • 3. break;는 switch 블록 전체를 빠져나갑니다. 빠뜨리면 아래 case가 연달아 실행되므로 매우 중요합니다! 반복문에서의 break는 나중에 더 자세히 배우므로, 지금은 switch-case를 종료한다는 것만 기억하세요.
  • 4.default: 어떤 case와도 일치하지 않을 때 실행됩니다. if-else의 else와 같은 역할입니다.

switch-case 문 상세 순서도

순차적 동등 비교: 위에서부터 변수값이 `case`의 값과 정확히 같은지(==) 검사합니다.
break 문의 역할: 코드가 실행된 후 break;를 만나면 파란색 경로를 타고 switch 문 전체를 즉시 빠져나갑니다. 만약 break가 없다면 아래쪽 case의 코드까지 연속해서 실행(Fall-through)됩니다.
default 블록: 앞선 모든 case와 일치하지 않을 때 실행되는 최후의 보루(if 문의 else와 같은 역할)입니다.

📥 아래 예제에서 처음 등장하는 것: scanf()와 &

scanf("%d", &score)는 키보드로 정수를 입력받아 score 변수에 저장하는 함수입니다.
printf가 화면에 출력한다면, scanf는 키보드로 입력받습니다.

&score에서 &는 "score 변수가 메모리에서 저장된 위치(주소)"를 의미합니다. scanf에게 "이 주소에 입력값을 저장해 줘"라고 알려주는 것입니다. 추후 포인터 단원에서 다시 학습할 개념이므로 지금은 "scanf를 쓸 때는 변수 앞에 항상 &를 붙인다"고 규칙처럼 알아두면 됩니다.

예제 간단한 switch문 — 요일 출력 (simple_switch_case.c)
simple_switch_case.c 
#include <stdio.h>
int main()
{
    int day = 2;

    /* day의 값에 따라 해당 case로 바로 이동합니다 */
    switch (day)
    {
        case 1:
            printf("월요일입니다.\n");
            break;  /* break를 만나야 switch문을 빠져나갑니다 */
        case 2:
            printf("화요일입니다.\n");
            break;
        case 3:
            printf("수요일입니다.\n");
            break;
        default:
            printf("1~3 사이의 숫자만 입력해주세요.\n");
    }

    return 0;
}
실행 예시 (2 입력 시)
화요일입니다. // day가 2이므로 case 2 실행 후 break로 종료
예제 switch문 — 학점 계산 (101.c)
101.c 
#include <stdio.h>
int main()
{
    int score;
    char grade;
    puts("점수를 입력하세요(1~100)");  /* 문자열 출력 후 자동 줄바꿈 */
    putchar('>');                      /* '>' 문자 1개 출력 (입력 프롬프트) */
    scanf("%d", &score);            /* 키보드로 정수 입력 → score에 저장 */

    /* score / 10 : 정수 나눗셈 → 소수점 이하 버림 */
    /* 예) 95 / 10 = 9,  85 / 10 = 8,  73 / 10 = 7  */
    switch (score / 10)
    {
      case 10:             /* 100점: break 없음 → 바로 아래 case 9로 fall-through */
      case 9: grade = 'A'; break;  /* 90~100점 */
      case 8: grade = 'B'; break;  /* 80~89점  */
      case 7: grade = 'C'; break;  /* 70~79점  */
      case 6: grade = 'D'; break;  /* 60~69점  */
      default: grade = 'F';        /* 59점 이하 */
    }
    printf("학점은 %c 입니다.", grade);
    return 0;
}
실행 예시 (85점 입력 시)
점수를 입력하세요(1~100) >85 학점은 B 입니다. // 85 / 10 = 8 → case 8 실행 → grade = 'B' → 출력

핵심 원리: score / 10은 어떻게 동작하나요?

C언어에서 정수 ÷ 정수 = 정수(몫만, 소수점 버림)입니다. 따라서 score / 10은 점수의 십의 자리 숫자를 꺼내는 역할을 합니다.

입력 점수 score / 10 결과 매칭 case 학점
100 100 / 10 = 10 case 10 → case 9 (fall-through) A
95 95 / 10 = 9 case 9 A
85 85 / 10 = 8 case 8 B
55 55 / 10 = 5 default F

코드 분석: case 10: 옆에는 왜 아무것도 없죠?

switch문은 break;를 만나기 전까지 아래로 계속 떨어지며 실행되는 성질 (Fall-through, 폴스루)가 있습니다.

case 10: 옆에 코드가 없고 break도 없으므로, 100점(10)이 입력되면 곧바로 case 9의 코드로 떨어져서 grade = 'A';를 실행합니다. 100점과 90점대를 같은 학점으로 처리하기 위해 의도적으로 break를 생략한 것입니다.

Section 9

선택 구조의 마술사 — 조건(삼항) 연산자

간단한 if-else 문은 단 한 줄로 줄여서 쓸 수 있습니다. 조건식, 참일 때의 값, 거짓일 때의 값 이렇게 3가지(피연산자 3개)가 필요하다고 해서 삼항 연산자라고도 부릅니다. 코드를 매우 간결하게 만들어주어 실무에서 자주 사용됩니다.

기본 문법

조건식 ? 참일 때 값 : 거짓일 때 값;

💡 "조건식이 참인가요? 참이면 이것: 거짓이면 저것" 으로 소리 내어 읽으면 쉽습니다.

일반적인 if-else 문 (5줄)

int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

삼항 연산자로 압축 (1줄)

int max = (a > b) ? a : b;
아래 예제에서 처음 등장하는 %s는 문자열(글자 묶음)을 출력하는 자리 표시자입니다.
삼항 연산자의 결과로 "짝수" 또는 "홀수"라는 문자열이 반환되고, 그 문자열이 %s 자리에 들어가 출력됩니다.
예제 삼항 연산자 — 짝수/홀수 판별
ternary.c 
#include <stdio.h>

int main() {
    int num = 15;

    /* 조건식: (num % 2 == 0) 이 참인가?           */
    /* 참이면 "짝수", 거짓이면 "홀수" 문자열 반환   */
    /* %s 자리에 그 문자열이 들어가 출력됩니다      */
    printf("%d는(은) %s입니다.\n", num, (num % 2 == 0) ? "짝수" : "홀수");

    return 0;
}
실행 결과
15는(은) 홀수입니다. // 15 % 2 == 0 ? → 15 ÷ 2 나머지 1 → 0이 아님 → 거짓 → "홀수" 출력

선택 구조 핵심 요약

  • C언어의 참과 거짓 — 숫자 0과 그 외 C언어는 숫자 0은 거짓(False)으로, 0이 아닌 모든 숫자(1, -1 등)는 참(True)으로 판단합니다.
  • if 계열 구문 구분 단순 if: 조건이 참일 때만 실행  |  if-else: 참/거짓 두 가지 중 하나  |  if-else if: 여러 조건을 순서대로 검사  |  중첩 if: 조건 안에 또 다른 조건
  • 상황에 맞는 구문 선택 — if vs switch 범위 조건이나 복잡한 논리는 if-else 문을, 정수나 문자의 정확한 값에 따른 단순 분기는 switch 문을 사용하는 것이 효율적입니다.
  • switch 문의 break 필수 사용 break 문이 없으면 아래쪽 case들이 연달아 실행되는 Fall-through가 발생합니다. 의도하지 않은 동작을 막으려면 반드시 작성해야 합니다.
  • 삼항 연산자 — 한 줄짜리 if-else 조건식 ? 참일때값 : 거짓일때값 형태로 간단한 if-else를 한 줄로 대체합니다.
