Chapter 2. C 프로그래밍의 기초

2. 입출력 함수

컴퓨터와 사용자 사이의 소통 창구인 표준 입출력 함수를 배우고, 데이터를 원하는 형식으로 제어하는 방법을 익힙니다.

Think & Objective

"형식을 지정하여 입력과 출력을 하는 함수는 어떻게 사용할까?"

  • putchar, puts, printf의 차이를 이해하고 상황에 맞게 쓸 수 있다.
  • getchar, scanf를 사용해 키보드 입력을 변수에 저장할 수 있다.
  • 서식 지정자와 이스케이프 시퀀스를 활용해 출력 형식을 제어할 수 있다.
  • 입력 버퍼 문제를 이해하고 스스로 해결할 수 있다.

먼저 전체 그림을 봅시다

이 장에서 배울 모든 함수는 아래 흐름의 한 부분입니다

입출력 함수를 낱개로 외우기 전에, 먼저 컴퓨터가 데이터를 주고받는 전체 흐름을 파악합니다. 아래 다이어그램을 이해하면 각 함수가 어떤 역할인지 즉시 알 수 있습니다.

Section 1

문자·문자열 출력: putchar( ), puts( )

화면에 결과를 보여주는 대표 함수는 printf입니다. 하지만 단순히 문자나 문장만 출력할 때는 더 가볍고 쓰기 편한 전용 도구가 있습니다.

putchar — "딱 한 글자만!"

오직 문자 1개(작은따옴표 ' ')만 출력합니다. 출력 후 줄바꿈이 없으므로 줄을 바꾸려면 putchar('\n');을 따로 써주어야 합니다.

putchar('A'); // 문자 직접

putchar(65); // ASCII 번호

putchar(ch); // char 변수

puts — "문장 + 자동 줄바꿈!"

여러 글자로 이루어진 문장(큰따옴표 " ")을 한 번에 출력합니다. 출력 후 자동으로 줄바꿈이 됩니다. printf("\n")을 따로 쓸 필요가 없습니다!

puts("Hello!"); // ← 자동 줄바꿈

puts("C언어!"); // ← 또 줄바꿈

개념 확인 putchar vs puts 차이점 확인하기
print_basic.c 
#include <stdio.h>

int main() {
    // 1. putchar 사용 (작은따옴표, 한 글자씩)
    putchar('H');
    putchar('i');
    putchar('\n'); // 직접 줄바꿈 기호를 넣어줘야 함

    // 2. puts 사용 (큰따옴표, 자동 줄바꿈)
    puts("Hello!");       // 출력 후 자동으로 줄바꿈
    puts("C언어 재미있네요."); // 출력 후 자동으로 줄바꿈

    return 0;
}
Hi Hello! C언어 재미있네요.

Section 2

컴퓨터는 문자를 어떻게 기억할까? (아스키 코드)

실습 1-1에서 putchar(105);에 숫자를 넣었는데 'i'가 출력되어 당황하셨나요? 컴퓨터는 글자를 있는 그대로 기억하지 않습니다. 대신 "A는 65번, a는 97번으로 부르자!"는 전 세계적인 약속이 있는데, 이를 아스키(ASCII) 코드라고 합니다.

꼭 알아야 할 아스키 코드

  • 'A'
    65번 ~ 90번 (A~Z)
  • 'a'
    97번 ~ 122번 (a~z)
    대문자보다 정확히 32가 큽니다!
  • '0'
    48번 ~ 57번 (0~9)
    ⚠ 아라비아 기호라서 Zero를 뜻하는 숫자 값 0과 다름!

핵심: 아스키 코드를 외울 필요는 없습니다! "컴퓨터 내부에서 문자는 숫자로 저장된다"는 사실만 기억하세요. 이 원리가 나중에 암호화 프로그램(실습 6-1)에 직접 활용됩니다.

개념 확인 대소문자 변환의 비밀
ascii_secret.c 
#include <stdio.h>
int main() {
    char ch = 'a'; // 소문자 a (97)

    // 97 - 32 = 65 → 'A' 출력!
    putchar(ch - 32);  // 대문자 변환
    putchar('\n');

    putchar('A' + 32); // 'A'(65) + 32 = 97 → 'a' 출력!
    return 0;
}

a

Section 3

형식 지정 출력: printf( )

printf()는 "Print Formatted"의 줄임말로, 변수의 값을 원하는 형식에 맞춰 화면에 출력합니다. putchar/puts와 달리 숫자, 소수, 문자 등 모든 자료형을 출력할 수 있습니다.

서식 지정자 완전 정리

%d

정수 (int) — 10진수

d = decimal (10진수)

printf("%d", 42)
%f

실수 (float)

f = floating-point (부동소수점)

printf("%.2f", 3.14)
%lf

실수 (double) — 더 정밀

l = long, f = floating-point

printf("%.3lf", pi)
%c

문자 하나 (char)

c = character

printf("%c", 'A')
%s

문자열 (char 배열) ← 중요!

s = string

printf("%s", name)
%o / %x

8진수 / 16진수

o = octal, x = hexadecimal

printf("%x", 255)

이스케이프 시퀀스 (제어 문자)

백슬래시(\)로 시작하는 특수 문자입니다. 화면에 보이지 않는 동작을 수행합니다. 대부분의 문자는 영어 단어의 약자입니다.

기호 의미 설명
\n 줄바꿈 n = new line (새 줄로 이동)
\t 탭 t = tab (일정 간격 이동)
\\ 백슬래시 출력 \ 문자를 그대로 출력
\" 큰따옴표 출력 문자열 내부에서 " 출력
\0 널 문자 문자열의 끝을 나타내는 문자 (ASCII 0)

Section 4

형식 지정 입력: scanf( )

scanf()는 "Scan Formatted"의 약자로, 키보드에서 데이터를 입력받아 변수에 저장합니다. printf()의 반대 역할입니다.

초보자 최빈 실수! — & 연산자를 반드시 붙여야 합니다

scanf("%d", &age); ✅
scanf("%d", age); ❌

& 기호는 "이 변수가 저장된 메모리 주소"를 의미합니다. Chapter 2-1에서 배운 것처럼, 변수 age는 메모리의 특정 방(예: 0x1000)에 저장되어 있습니다. scanf는 그 방 주소를 알아야 값을 정확히 넣어줄 수 있습니다.

scanf("%d", &age) 실행 시

scanf()
"값 배달원"
&age
= 주소 0x1000
"배달 주소"
age
20

0x1000 ~ 0x1003 (int, 4Bytes)

입력한 20이 age 메모리 방에 저장됨!

Step 1
정수 하나 입력받기
scanf_basic.c 
#include <stdio.h>

int main() {
    int num;
    printf("숫자를 입력하고 엔터를 치세요: ");
    scanf("%d", &num);  // & 필수! num의 메모리 주소로 값을 전달
    printf("당신이 입력한 숫자는 %d입니다.\n", num);
    return 0;
}
실행 예시:
숫자를 입력하고 엔터를 치세요: 42↵
당신이 입력한 숫자는 42입니다.
Step 2
여러 변수 동시에 입력받기
scanf_multi.c 
#include <stdio.h>

int main() {
    int height, weight;
    printf("키와 몸무게를 입력하세요 (띄어쓰기로 구분): ");
    scanf("%d %d", &height, &weight); // 두 개 동시 입력! 각각 & 필수
    printf("키는 %dcm이고 몸무게는 %dkg입니다.\n", height, weight);
    return 0;
}
실행 예시:
키와 몸무게를 입력하세요 (띄어쓰기로 구분): 175 68↵
키는 175cm이고 몸무게는 68kg입니다.
💡 %d %d처럼 서식 지정자 사이에 공백을 넣으면 스페이스나 엔터로 구분된 두 값을 한 번에 입력받을 수 있습니다.

Section 7

C언어 최대의 함정: 엔터키의 배신 (입력 버퍼)

scanf로 숫자를 입력하고 [Enter]를 치면, 숫자는 변수로 들어가지만 엔터키(\n) 자체는 입력 버퍼(대기실)에 몰래 남아있습니다. 그 직후에 문자 1개를 입력받는 코드가 실행되면, 컴퓨터는 이 남은 엔터키를 문자 입력으로 착각하고 사용자에게 입력 기회도 주지 않고 넘어가 버립니다!

찌꺼기를 처리하는 마법의 공백

%c 앞에 공백(스페이스) 하나를 넣으면 버퍼에 남아있는 엔터, 스페이스 등 공백 문자를 모두 무시하고 진짜 의미있는 문자만 가져옵니다.

scanf("%c", &blood); ❌

scanf(" %c", &blood); ⭕ ← % 앞 공백 주목!

공백의 역할: "%c 앞의 공백은 버퍼에 남아있는 엔터, 스페이스 등 모든 공백 문자를 자동으로 무시해 줍니다."

버퍼(대기실)의 상태 변화
1단계 나이 17 입력 후 Enter(\n)
1
7
\n

↑ 17은 변수로, \n은 버퍼에 남음!

문제 scanf("%c")가 남은 \n을 혈액형으로 착각!
해결 scanf(" %c")가 \n을 무시하고 다음 문자 대기!
버그 코드
혈액형이 건너뛰어짐!
scanf("%d", &age);
// age에 17이 저장, \n이 버퍼에 남음
scanf("%c", &blood); // \n을 읽어버림!
// blood에 '\n'이 저장 → 혈액형 입력 불가!
해결 코드
% 앞 공백 하나로 해결!
scanf("%d", &age);
// age에 17이 저장, \n이 버퍼에 남음
scanf(" %c", &blood); // 공백이 \n 무시!
// blood에 사용자가 입력한 'A' 등이 정상 저장!

Section 8

문자열 입력: gets( ) 와 fgets( )

한 글자가 아니라 긴 단어나 문장(공백 포함)을 통째로 입력받을 때 사용합니다.

gets() 보안 경고 — 교과서와 실무의 차이

최신 컴파일러(Visual Studio 등)는 gets()가 보안 취약점(버퍼 오버플로우)을 가지므로 사용을 금지하고 경고를 표시합니다. 교과서 코드를 그대로 실행하면 빨간 경고가 뜰 수 있지만 학습 목적이므로 무시해도 됩니다. 실제 프로젝트에서는 반드시 fgets()를 사용하세요!

#include <stdio.h>
int main()
{
    char s[100];
    printf("이름을 입력하세요: ");
    gets(s);    // ⚠ 보안 경고 있음! 하지만 학습용으로 사용
    puts(s);    // 입력한 문자열 출력
    return 0;
}
#include <stdio.h>
int main()
{
    char s[100];
    printf("이름을 입력하세요: ");
    fgets(s, sizeof(s), stdin); // ✅ 안전한 버전!
    //  ^배열  ^최대크기  ^키보드 입력
    printf("입력한 이름: %s", s);
    return 0;
}
실행 예시:
이름을 입력하세요: 홍길동↵
홍길동
💡 fgets(배열명, 최대크기, stdin): stdin은 "표준 입력(키보드)"을 의미합니다. sizeof(s)로 배열 크기를 자동으로 전달하는 것이 안전합니다.

#include <stdio.h>
int main()
{
    int key;
    char secret;

    printf("문자를 입력하세요: ");
    secret = getchar(); // 문자 1개 입력

    // getchar() 후 버퍼에 남은 \n 제거!
    getchar(); // 엔터 소모용

    printf("키 값을 입력하세요: ");
    scanf("%d", &key);

    // 대문자로 변환(-32) 후 key만큼 이동!

Section 10

종합 실습: 나의 첫 번째 대화형 프로그램

이 장에서 배운 putchar, puts, printf, getchar, fgets, scanf, 버퍼 처리를 모두 사용하는 완성형 프로그램입니다. 사용자의 프로필을 입력받아 출력합니다.

user_profile.c 
#include <stdio.h>

int main() {

    // 변수 선언
    char name[100];
    int age;
    char bloodType;
    double height;

    // ① 이름 입력 (문자열 — fgets 사용)
    puts("=== 나의 프로필 입력 ===");
    puts("이름을 입력하세요:");
    putchar('>');
    putchar(' ');
    fgets(name, sizeof(name), stdin); // 안전한 문자열 입력

    // ② 나이 입력 (정수 — scanf 사용)
    puts("나이를 입력하세요:");
    putchar('>');
    putchar(' ');
    scanf("%d", &age);

    // ③ 혈액형 입력 (문자 — " %c"로 버퍼 문제 해결!)
    puts("혈액형을 입력하세요 (A/B/O/AB 중 첫 글자):");
    putchar('>');
    putchar(' ');
    scanf(" %c", &bloodType); // " %c" — 앞 공백으로 \n 제거!

    // ④ 키 입력 (실수 — scanf %lf 사용)
    puts("키를 입력하세요 (cm):");
    putchar('>');
    putchar(' ');
    scanf("%lf", &height);

    // ⑤ 결과 출력 (printf + 서식 지정자)
    printf("\n=== 입력하신 프로필 ===\n");
    printf("이름    : %s", name);    // fgets는 \n 포함하므로 따로 \n 불필요
    printf("나이    : %d세\n", age);
    printf("혈액형  : %c형\n", bloodType);
    printf("키      : %.1lf cm\n", height);

    return 0;
}
실행 예시: === 나의 프로필 입력 === 이름을 입력하세요: > 홍길동 나이를 입력하세요: > 20 혈액형을 입력하세요 (A/B/O/AB 중 첫 글자): > A 키를 입력하세요 (cm): > 175.5
=== 입력하신 프로필 === 이름 : 홍길동 나이 : 20세 혈액형 : A형 키 : 175.5 cm

출력 함수

putchar / puts / printf

입력 함수

getchar / fgets / scanf

서식 지정자

%d %f %lf %c %s

버퍼 처리

scanf(" %c", &x)

Self-Check

자가점검 — 스스로 답할 수 있나요?

아래 질문에 책이나 노트를 보지 않고 답할 수 있으면 이 장의 학습이 완료된 것입니다.

Q1

putchar()와 puts()의 차이점은 무엇인가요?

힌트: 출력 단위, 줄바꿈 자동 여부

Q2

scanf()에서 변수 앞에 & 를 붙여야 하는 이유는 무엇인가요?

힌트: 메모리 주소, 직접 저장

Q3

putchar(65)를 실행하면 어떤 결과가 나오나요? 왜 그런가요?

힌트: ASCII 코드, 'A'의 번호

Q4

나이를 scanf로 입력받은 뒤 혈액형을 입력받을 때 scanf("%c")가 실패하는 이유와 해결법은?

힌트: 입력 버퍼, \n 찌꺼기, " %c"

Q5

printf("%.2lf", 3.14159)의 출력 결과는 무엇인가요?

힌트: 소수점 자리수 제어

Q6

gets()대신 fgets()를 사용해야 하는 이유는 무엇인가요?

힌트: 버퍼 오버플로우, 보안

정답 확인 (클릭)

A1. putchar()는 문자 1개만 출력하며 줄바꿈 없음. puts()는 문자열 전체를 출력하고 자동으로 줄바꿈.

A2. &는 변수의 메모리 주소를 전달합니다. scanf는 직접 그 주소로 찾아가 값을 저장하기 때문에 주소가 없으면 어디에 넣어야 할지 모릅니다.

A3. 'A'가 출력됩니다. ASCII 코드 65번이 대문자 'A'이기 때문입니다.

A4. scanf("%d") 실행 후 엔터(\n)가 입력 버퍼에 남아, 다음 scanf("%c")가 그 \n을 읽어버립니다. 해결: scanf(" %c") — % 앞의 공백이 버퍼 찌꺼기를 무시합니다.

A5. 3.14 (소수점 둘째 자리까지 출력)

A6. gets()는 입력 크기를 제한하지 않아 배열 범위를 초과하는 버퍼 오버플로우 보안 취약점이 있습니다. fgets(s, sizeof(s), stdin)은 최대 크기를 지정하여 안전합니다.
