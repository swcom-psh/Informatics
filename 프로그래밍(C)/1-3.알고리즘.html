Chapter 1. 프로그래밍의 개요

1-3. 알고리즘

어떤 문제를 해결하기 위한 명확하고 효율적인 단계적 절차, 알고리즘의 개념과 다양한 표현 방법을 알아봅니다.

Think & Objective

"양팔 저울을 이용하여 40mg의 추를 가장 적은 비교 횟수로 찾는 방법은 무엇일까요?"

  • 주어진 문제를 해결하기 위한 알고리즘의 개념을 설명할 수 있다.
  • 순서도 또는 의사 코드 등을 이용하여 문제를 해결하는 알고리즘을 작성할 수 있다.

1. 알고리즘의 개념과 5가지 조건

알고리즘(Algorithm)이란 어떤 일을 해결하기 위한 방법이나 문제를 해결하기 위한 절차 등을 단계적으로 나열한 것입니다. 효율적으로 설계된 알고리즘은 보다 짧은 시간에, 보다 많은 작업을 더 빠르고 정확하게 수행할 수 있게 해줍니다.

올바른 알고리즘이 되기 위한 5가지 조건

1. 입력 (Input)

외부에서 제공되는 자료가 0개 이상 존재해야 합니다.

2. 출력 (Output)

적어도 1개 이상의 결과를 만들어 내야 합니다.

3. 명확성 (Definiteness)

각 단계의 명령은 단순하고 모호하지 않아야 합니다.

4. 유한성 (Finiteness)

명령어들을 수행한 후에는 반드시 종료되어야 합니다.

5. 수행 가능성 (Effectiveness)

모든 명령은 실행 가능한 것이어야 합니다.

2. 알고리즘의 3가지 표현 방법

알고리즘은 누구나 이해하기 쉽게 표현해야 합니다. 일상 언어부터 도형까지 다양한 방식으로 표현할 수 있습니다.

1. 자연어 (Natural Language)

우리가 일상생활에서 사용하는 언어를 이용하여 문제 해결 과정을 순서대로 서술하는 방법입니다. 누구나 쉽게 이해할 수 있지만, 문장이 길어지거나 의미가 모호해질 수 있습니다.

예) "두 수 중 큰 수에서 작은 수를 뺀 결과를 출력해라."

2. 의사 코드 (Pseudocode)

자연어보다 간략하고, 실제 프로그래밍 언어의 문법과 유사하게 논리적인 순서로 표현하는 방법입니다. 코드로 변환하기 매우 쉽습니다.

입력: A, B
만약 A > B 이면 Result = A - B
아니면 Result = B - A
출력 Result

3. 순서도 (Flowchart)

약속된 기호와 도형, 화살표를 이용하여 논리적 흐름을 시각적으로 나타내는 방법입니다. 전체적인 흐름을 한눈에 파악하기 좋고 논리적 오류를 찾기 쉽습니다.

[타원] 시작/끝 ➔ [평행사변형] 입출력 ➔ [마름모] 조건 판단 ➔ [직사각형] 처리

3. 실습: 비만도(BMI) 계산 알고리즘

키와 몸무게를 입력받아 비만도(BMI)를 구하고, 결과에 따라 상태를 출력하는 알고리즘을 설계해 봅시다.
* BMI 공식 = 몸무게(kg) ÷ (키(m) × 키(m))

실습 의사 코드 및 파이썬 코드로 표현하기
의사 코드 (Pseudocode)

1. 키(h)와 몸무게(w)를 입력받는다.
2. 신체 질량 지수(BMI) = w / (h * h) 를 계산한다.
3. 만약 BMI가 25 이상이면 "비만"을 출력한다.
4. 그렇지 않다면 "정상"을 출력한다.

bmi_calculator.py 
# 키(m)와 몸무게(kg) 입력
h = float(input("키를 입력하세요(m): "))
w = float(input("몸무게를 입력하세요(kg): "))

# BMI 계산
bmi = w / (h * h)

# 조건에 따른 결과 출력
if bmi >= 25:
    print("비만입니다.")
else:
    print("정상입니다.")
