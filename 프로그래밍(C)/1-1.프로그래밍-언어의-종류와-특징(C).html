Chapter 1. 프로그래밍의 개요

1-1. 프로그래밍 언어의 종류와 특징

컴퓨터와 대화하기 위한 언어! 다양한 프로그래밍 언어의 특징을 이해하고, 개발 분야에 맞는 적절한 언어를 선택하는 방법을 알아봅니다.

Think & Objective

"프로그래밍 언어를 특징에 따라 용도에 맞게 선택하여 사용할 수 있을까?"

  • 각종 프로그래밍 언어의 종류와 특징을 설명할 수 있다.
  • 프로그램의 개발 분야와 용도에 따라 적절한 프로그래밍 언어를 선택할 수 있다.

1. 프로그램과 프로그래밍

프로그램 (Program)

주어진 문제를 컴퓨터로 처리하기 위해 작업의 순서와 방법을 지시하는 일련의 문장들의 집합체

프로그래밍 (Programming)

컴퓨터로 어떤 문제를 처리할 수 있도록 프로그램을 작성하는 작업이나 과정

프로그래머 (Programmer)

프로그램을 설계하고 프로그래밍 언어를 사용하여 명령어를 작성하는 사람

2. 주요 언어의 특징과 용도

세계에는 수많은 프로그래밍 언어가 존재합니다. 개발하려는 분야(웹, 앱, 게임, 인공지능 등)에 따라 가장 적합하고 효율적인 언어를 선택하는 것이 중요합니다.

C

C 언어

프로그램 크기가 작으며 실행 속도가 빠르고 메모리를 효과적으로 사용할 수 있습니다. 기계어 수준의 구체적인 하드웨어 제어가 가능하여 시스템 프로그래밍이나 임베디드 기기에 널리 사용됩니다.

Java

Java (자바)

같은 프로그램을 다양한 운영체제에서 그대로 사용할 수 있는 높은 이식성을 가집니다. 웹 프로그래밍, 안드로이드 모바일 앱 분야, 데이터베이스 관리 등 엔터프라이즈 환경에서 널리 사용됩니다.

Py

Python (파이썬)

문법이 간결하여 배우기가 쉽고 라이브러리(모듈)가 매우 풍부합니다. 대학, 교육 기관은 물론 연구 기관, 산업계 등에서 인공지능(AI), 데이터 분석, 웹 서버 개발 등 다방면으로 활용됩니다.

PHP

PHP

적은 명령어들로 프로그래밍이 가능하며, 웹 프로그램에서 HTML 코드 안에 직접 포함하여 사용할 수 있습니다. 서버 데이터베이스와의 연동이 매우 쉽다는 장점이 있습니다.

Sc

Scratch (스크래치)

프로그램을 처음 배우는 초보자에게 적합한 교육용 언어입니다. 다른 언어들이 텍스트 형식의 코딩을 하는 데 반해, 스크래치는 블록을 마우스로 끌어당겨 조립하는 방식으로 논리력을 키워줍니다.
