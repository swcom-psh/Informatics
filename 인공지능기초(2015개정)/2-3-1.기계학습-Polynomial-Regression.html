Multiple Linear Regression

세상의 많은 결과는 단 하나의 원인으로 결정되지 않습니다. 여러 개의 데이터를 조합하여 최적의 예측 공식을 찾아내는 과정을 직관적으로 확인해보세요.

다중선형회귀
도입 개념설명 시뮬레이터 퀴즈
Core Mechanism

여러 개의 원인으로 결과를 어떻게 예측할까요?

인공지능은 각 원인이 결과에 미치는 '영향력(가중치)'을 조절하며 최적의 공식을 만들어냅니다.

Step 1

다중 데이터 수집

결과(y)에 영향을 줄 수 있는 여러 가지 원인 데이터(x₁, x₂...)를 모아 준비합니다.

Step 2

초기 공식 생성

각 원인이 얼마나 중요한지 모르는 상태이므로, 처음에는 무작위로 가중치(영향력)를 설정합니다.

Step 3

오차 계산

만들어진 공식으로 예측한 결과와 실제 정답 데이터를 비교하여 얼마나 틀렸는지(오차) 계산합니다.

Step 4

가중치 수정 (학습)

오차를 줄이는 방향으로 더 중요한 원인의 가중치는 키우고, 덜 중요한 것은 줄여나갑니다.

컴퓨터가 생각하는 예측 공식의 형태

결과(y) = 가중치1(w₁) × 원인1(x₁) + 가중치2(w₂) × 원인2(x₂) + 기본값(b)

다중선형회귀는 입력된 데이터들을 바탕으로 위 공식에 들어갈 가장 완벽한 숫자(가중치 w와 기본값 b)를 스스로 찾아내는 과정입니다.

핵심 정리

  • 다중선형회귀는 2개 이상의 원인(독립 변수)을 사용하여 결과를 예측합니다.
  • 가중치(w)는 해당 변수가 결과에 미치는 영향력의 크기를 의미합니다.
  • 가중치가 양수(+)면 결과도 증가하고, 음수(-)면 결과는 감소하는 경향을 보입니다.

주의할 점 (오개념 방지)

  • 원인(변수)의 개수가 무조건 많다고 해서 항상 예측이 정확해지는 것은 아닙니다. 불필요한 정보는 오히려 방해가 될 수 있습니다(과적합 문제).
  • 두 변수가 함께 변한다고 해서 반드시 하나가 다른 하나의 '직접적인 원인'인 것은 아닙니다(상관관계 ≠ 인과관계).
Interactive Simulator

다중선형회귀 학습 시뮬레이터

결과에 영향을 미치는 두 가지 원인(x₁, x₂)의 부분 의존성(Partial Dependence)을 그래프로 확인하세요.

원인1(x₁)에 따른 결과(y)
* x₂는 평균값으로 고정하여 추정한 선입니다.
원인2(x₂)에 따른 결과(y)
* x₁은 평균값으로 고정하여 추정한 선입니다.

시뮬레이터 가이드

  • 예제 버튼을 클릭하여 미리 준비된 실생활 데이터를 불러올 수 있습니다.
  • 직접 입력 시에는 x₁, x₂, y 값을 모두 채운 후 '추가' 버튼을 눌러주세요.
  • 랜덤 데이터 버튼은 현실적인 통계 분포를 묘사한 가상 데이터를 무작위로 생성합니다.
