컴퓨터의 감각 기관, '센서'

컴퓨터가 외부 환경을 지각하는 방법을 살펴봅시다.

사람이 눈, 귀, 피부로 세상을 느끼듯, 지능 에이전트(인공지능)는 '센서(Sensor)'를 통해 세상을 느낍니다.

우리 주변의 빛, 소리, 온도처럼 끊임없이 변하는 자연의 정보를 '아날로그 신호'라고 부릅니다. 센서는 이 자연의 아날로그 신호를 기계가 다룰 수 있는 '전기적 신호'로 바꿔주는 아주 중요한 역할을 합니다.

카메라 센서 빛(시각) 정보를 수집
(자율주행, 얼굴 인식)
마이크 센서 소리(청각) 정보를 수집
(인공지능 스피커)
지문 센서 표면의 굴곡을 수집
(스마트폰 잠금 해제)

세상을 컴퓨터의 언어로 번역하기 (A/D 변환)

A/D 변환기는 어떻게 통역할까? (3단계 마법)

A/D(Analog to Digital) 변환기는 끊어지지 않고 이어지는 아날로그 신호를 컴퓨터가 아는 0과 1로 바꾸기 위해 세 가지 과정을 거칩니다. 요리하는 과정에 비유해 볼까요?

1. 샘플링 (Sampling)

"잘게 쪼개기"
이어지는 신호를 일정한 시간 간격으로 탁! 탁! 끊어서 조각냅니다. (예: 긴 가래떡을 일정한 크기로 썰기)

2. 양자화 (Quantization)

"숫자 매기기"
쪼갠 조각들의 크기를 측정해서 가까운 정수(숫자)로 값을 매깁니다. (예: 썰어둔 떡의 무게를 저울로 달아 적기)

3. 부호화 (Encoding)

"0과 1로 쓰기"
숫자로 매긴 값을 컴퓨터만 읽을 수 있는 0과 1의 이진수로 최종 번역합니다. (예: 무게를 컴퓨터 문서로 타자 치기)

Deep Dive

왜 양자화가 중요한가요?

아날로그 신호는 무한한 정밀도를 갖지만, 컴퓨터는 이를 저장하기 위해 유한한 눈금을 사용해야 합니다. 이때 실제 값과 눈금의 차이인 양자화 오차가 발생합니다.

비트(Bit) 수 ↑

눈금이 촘촘해짐, 오차 감소, 원본에 가까움 (고음질/고화질)

비트(Bit) 수 ↓

눈금이 듬성듬성함, 오차 증가, 데이터 용량 작음

양자화 비트 수에 따른 차이 시각화

2-bit (4단계 눈금) 오차 큼
4-bit (16단계 눈금) 오차 작음

센서를 100% 믿어도 될까?

센서의 오작동을 대비해야 한다!

자주 하는 오해

"카메라 센서는 사람의 눈처럼 사물을 완벽하게 알아본다?"

정답은 '아니요' 입니다! 사람은 빛이 강하든 그림자가 지든 사과를 사과로 알아봅니다. 하지만 센서는 '빛의 변화'에 매우 취약합니다. 빛이 조금만 반사되거나 역광이 비치면 아예 다른 물체로 착각하거나 인식하지 못하는 '인식 오류'가 발생할 수 있습니다.

빛의 마술과 자율주행차

앞에 트럭이 있어도, 강한 햇빛이 하얗게 반사되면 카메라 센서는 눈이 부셔서 트럭을 빈 공간으로 인식할 수 있습니다. 그래서 자율주행 자동차는 카메라 하나만 믿지 않고 레이더, 라이다 등 여러 센서를 동시에 사용합니다.

핵심: 센서에만 100% 의존할 수는 없다!

컴퓨터 내부에서 진행되는 인식의 과정

Under the hood

컴퓨터 프로그램 속에서는 인식 과정이 어떻게 진행될까?

지금까지 배운 환경 정보 → 센서 → A/D 변환 → 인식 과정은 실제 프로그램에서도 비슷한 순서로 처리됩니다.

오른쪽 코드를 보며 아날로그 소리 데이터가 0과 1의 디지털 데이터로 변환되는 과정을 단계별로 확인해 보세요.

sensor_process.c 

// 에이전트가 소리를 듣고 인식하는 과정

void agentPerception() {
    // [1단계: 환경 정보] 자연의 소리(아날로그) 발생
    AnalogSignal soundWave = environment.getSound();

    // [2단계: 센서] 마이크가 소리를 '전기 신호'로 바꿈
    ElectricSignal rawSignal = micSensor.detect(soundWave);

    // [3단계: A/D 변환기] 컴퓨터의 통역사 등장!
    // 전기 신호를 0과 1의 '디지털 데이터'로 번역함
    DigitalData dData = adConverter.transform(rawSignal);

    // [4단계: 인식] 에이전트의 두뇌가 디지털 데이터를 분석함
    if (aiBrain.recognizeVoice(dData) == "도와줘") {
        actuator.moveTowardUser(); // 사용자에게 다가감
    }
}
