딥러닝(Deep Learning)이란 무엇일까?

딥러닝은 데이터 속에 숨겨진 규칙이나 패턴을 기계가 스스로 찾아내게 하는 '기계학습(Machine Learning)'의 한 방법입니다. 가장 큰 특징은 우리의 뇌 신경망 구조를 모방하여 복잡한 네트워크 형태의 모델을 만든다는 것입니다.

💡 입력층과 출력층 사이에 숨겨져 있는 '은닉층(Hidden Layer)'이 1개 이상 존재할 때, 우리는 이것을 심층 신경망(DNN: Deep Neural Networks)이라고 부릅니다. 은닉층이 '깊게(Deep)' 쌓여 있어서 '딥러닝'이라는 이름이 붙었답니다!

딥러닝의 장점

  • 기존 방법보다 훨씬 더 정확한 모델을 만들 수 있습니다.
  • 복잡한 문제를 해결하는 유용한 모델을 만드는 시간을 단축할 수 있습니다.

딥러닝의 단점

  • 수많은 계산을 해야 하므로 아주 높은 컴퓨터 성능(GPU 등)이 필요합니다.
  • 결과가 어떻게 나왔는지 과정을 설명하기 어려운 '블랙박스' 문제가 있습니다.

딥러닝의 세포, '퍼셉트론'

신경망을 이루는 가장 작은 단위를 '퍼셉트론(Perceptron)'이라고 합니다. 퍼셉트론은 여러 데이터를 받아들여 중요도(가중치)를 곱한 뒤, 그 합이 특정 기준을 넘으면 결과를 출력합니다.

인간의 뇌 신경세포(뉴런)를 모방하다

퍼셉트론은 우리 뇌를 구성하는 신경세포인 '뉴런(Neuron)'의 작동 방식을 수학적으로 본떠 만든 것입니다. 생물학적 뉴런이 전기 신호를 주고받아 판단을 내리듯, 인공 퍼셉트론은 숫자(데이터)를 계산하여 결과를 만들어냅니다.

🧠 생물학적 뉴런

  • 수상돌기: 다른 뉴런들로부터 신호를 받아들임
  • 시냅스: 신호의 연결 강도(중요도)를 결정함
  • 세포체: 들어온 신호들을 모아 판단함
  • 축삭돌기: 다음 뉴런으로 신호를 전달함

💻 인공 퍼셉트론

  • 입력값: 외부 데이터나 이전 층의 값을 받아들임
  • 가중치: 각 데이터의 중요도를 곱함
  • 활성화: 값을 모두 더하고 출력 여부를 결정함
  • 출력값: 다음 퍼셉트론으로 결과를 전달함

예제: 컴퓨터는 고양이 사진을 어떻게 알아볼까?

딥러닝은 사진을 한 번에 보고 "고양이네!"라고 아는 것이 아닙니다. 단계별(은닉층)로 특징을 뽑아내어 조립합니다.

  1. 입력층: 사진의 수많은 픽셀(점) 데이터를 입력받습니다.
  2. 은닉층 1: 선, 모서리, 색상 같은 단순한 특징을 찾아냅니다.
  3. 은닉층 2: 선들을 조립해 눈, 코, 귀 모양 같은 부분적인 특징을 만듭니다.
  4. 은닉층 3: 부분들을 조립해 전체적인 고양이의 얼굴 형태를 완성합니다.
  5. 출력층: 최종적으로 이 사진이 고양이일 확률을 계산하여 결과를 출력합니다.

💻 코드 예제로 보는 퍼셉트론 원리

※ 개념 이해를 돕기 위해 입력값에 가중치를 곱해 결과를 판단하는 과정을 표현한 의사코드(Pseudo Code)입니다.

perceptron_logic.c 

// 고양이 귀 모양(입력1)과 수염 모양(입력2) 데이터
int input_ear = 80;
int input_whisker = 60;

// 각 특징에 대한 중요도(가중치) 설정
float weight_ear = 0.6;   // 귀 모양이 더 중요함
float weight_whisker = 0.4;

// 1. 입력값에 가중치를 곱하여 모두 합산합니다.
float total_score = (input_ear * weight_ear) + (input_whisker * weight_whisker);
// 계산: (80 * 0.6) + (60 * 0.4) = 48 + 24 = 72

float threshold = 70.0;   // 고양이라고 판단하는 기준 점수

// 2. 기준 점수를 넘는지 판단합니다. (활성화 함수 역할)
if (total_score >= threshold) {
    printf("고양이일 확률이 높습니다!\n");
} else {
    printf("고양이가 아닌 것 같습니다.\n");
}

퍼셉트론의 첫 번째 위기: XOR 문제

초기 퍼셉트론은 AND나 OR 같은 단순한 논리 회로를 완벽하게 해결하며 기대를 모았습니다. 하지만 XOR(배타적 논리합) 문제라는 벽에 부딪히게 됩니다. XOR는 입력된 두 값이 서로 다를 때만 1(참)이 되는 문제인데, 직선 하나로는 정답과 오답을 가를 수 없었기 때문입니다.

이 난관을 극복한 방법이 바로 '은닉층(Hidden Layer)'을 추가하는 것이었습니다. 퍼셉트론 여러 개를 겹쳐서 사용(다층 퍼셉트론)하면, 직선으로만 판단하던 인공지능이 공간을 왜곡하거나 여러 번 선을 긋는 효과를 내어 복잡한 XOR 정답을 찾아낼 수 있게 됩니다. 이것이 오늘날 우리가 부르는 딥러닝의 시작이 되었습니다.

퍼셉트론이 모이면? '심층 신경망(DNN)' 탄생!

앞서 살펴본 퍼셉트론 1개는 단순한 계산과 판단밖에 하지 못합니다. 하지만 이 퍼셉트론들을 수십, 수백 개씩 가로(층)와 세로(깊이)로 촘촘하게 연결하면 어떻게 될까요?

단일 퍼셉트론

간단한 참/거짓 판단

다층 퍼셉트론 (MLP)

은닉층 형성, 복잡한 패턴 인식

심층 신경망 (DNN)

은닉층이 '깊게(Deep)' 쌓인 형태

💡 딥러닝(Deep Learning)의 진짜 의미: 맨 처음 개념 섹션에서 보았던 '여러 개의 은닉층'이 바로 이 퍼셉트론들이 거물망처럼 겹겹이 쌓인 형태입니다. 층이 깊어질수록(Deep) 인공지능은 선이나 색상 같은 단순한 특징을 넘어, '고양이의 얼굴 형태' 같은 훨씬 고차원적이고 복잡한 특징을 스스로 조립하고 학습할 수 있게 됩니다.

오해하기 쉬운 딥러닝의 진실

"딥러닝 모델이 왜 이런 결과를 냈는지 개발자에게 물어보면 알 수 있나요?"
놀랍게도 알기 어렵습니다. 딥러닝은 수많은 은닉층과 가중치가 복잡하게 얽혀서 연산되기 때문에, 어떤 이유로 그런 결정을 내렸는지 인간이 완벽하게 추적하고 설명하기가 매우 어렵습니다. 이를 딥러닝의 '모델 해석의 어려움(블랙박스 문제)'이라고 합니다.

인공지능은 어떻게 스스로 정답을 찾을까?

가중치(중요도)를 스스로 조절하는 '학습'의 비밀

1. 부드러운 스위치, 활성화 함수

앞선 예제에서는 점수가 70점을 넘으면 무조건 '고양이(1)', 아니면 '아님(0)'으로 판단했습니다. 하지만 실제 세상의 문제는 이렇게 딱 잘라 말하기 어렵습니다. 그래서 딥러닝은 결과값을 부드럽게 만들어주는 '활성화 함수(Activation Function)'를 사용합니다.

계단 함수 (Step Function)

0 아니면 1! 단호하지만, 얼마나 확신하는지 알 수 없어요.

ReLU (렐루) 함수

요즘 가장 많이 쓰여요! 마이너스는 0으로, 플러스는 있는 그대로 전달합니다.

2. 오답 노트와 산길 내려오기 (경사하강법)

인공지능이 처음에 찍은 가중치(중요도)는 엉망진창입니다. 그래서 정답과 자신이 예측한 결과의 차이를 계산하는데, 이 오차를 '손실(Loss)'이라고 부릅니다. 인공지능의 목표는 이 손실을 최소화(0에 가깝게)하는 것입니다.

🏔️ 경사하강법 (Gradient Descent)

안대를 쓰고 산꼭대기에서 가장 낮은 골짜기(손실이 0인 곳)를 찾아간다고 상상해 보세요. 발바닥의 감각으로 '경사가 아래로 향하는 쪽'을 찾아 한 걸음씩 내려가는 방법입니다.

  • 손실 함수(Loss Function): 현재 내 위치(오차)가 얼마나 높은지 알려주는 고도계
  • 학습률(Learning Rate): 한 번에 얼마나 큰 보폭으로 걸어갈지 결정하는 발걸음 크기

💻 코드로 보는 '가중치 업데이트(학습)' 원리

인공지능 분야에서 가장 많이 쓰이는 파이썬(Python) 스타일의 코드로 학습 과정을 살펴봅시다. 오차를 줄이는 방향으로 가중치를 스스로 수정합니다.

gradient_descent.py 

def train_ai():
    weight = 0.5          # 인공지능이 무작위로 찍은 초기 가중치
    learning_rate = 0.1   # 발걸음 크기 (너무 크면 산을 넘어버림!)

    for step in range(100): # 100번 반복해서 산을 내려갑니다
        prediction = input_data * weight
        error = target_answer - prediction  # 오차(Loss) 계산: 정답과 얼마나 다른가?

        # ✨ 핵심: 오차에 비례해서 가중치를 수정합니다 (경사하강법)
        weight = weight + (error * learning_rate)

        if error == 0:
            print("학습 완료! 완벽한 가중치를 찾았습니다.")
            break
