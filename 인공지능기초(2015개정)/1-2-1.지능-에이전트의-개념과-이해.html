스스로 생각하고 행동하는
지능 에이전트

인공지능은 컴퓨터 화면 안에만 있을까요?
우리 삶 속에서 직접 움직이며 도움을 주는 '지능 에이전트'에 대해 알아봅시다.

지능 에이전트의 개념과 이해
개념 이해 작동 구조 3가지 특징 코드 예제 확인 퀴즈

인공지능과 에이전트의 관계

인공지능과 지능 에이전트는 서로 다른 개념이지만 함께 작동할 때 실제 시스템이 됩니다.

인공지능 (AI)

사람의 학습, 판단, 추론 능력을 컴퓨터가 수행하도록 만든 기술입니다. 즉 생각하고 판단하는 두뇌 역할을 합니다.

지능 에이전트

센서로 환경을 인식하고 행동을 수행하는 시스템입니다. 즉 환경과 상호작용하는 몸 역할을 합니다.

'인공지능'이 사람의 두뇌와 같다면, '에이전트'는 환경을 느끼고 움직이는 몸(프로그램 또는 기계)과 같습니다.

인공지능이 아무리 똑똑해도 생각만 하고 있다면 우리 생활에 도움을 줄 수 없습니다. 그래서 인공지능은 센서를 통해 환경을 인식하고 행동으로 결과를 만들어 내는 '지능 에이전트(Intelligent Agent)' 형태로 실제 시스템에 적용됩니다.

헷갈리지 마세요! (오개념 방지)

"지능 에이전트가 알아서 다 해주면 사람은 아무것도 안 해도 되나요?"
그렇지 않습니다. 지능 에이전트는 인간을 대신하는 존재가 아니라 인간의 목표 달성을 돕는 협력 도구입니다. 즉 사람과 함께 문제를 해결하는 파트너 역할을 합니다.

에이전트는 어떻게 세상을 이해하고 움직일까?

지능 에이전트가 임무를 수행하려면 사람의 몸과 비슷한 3가지 요소가 필요합니다.

① 센서 (Sensor)

"사람의 눈과 귀"

자신을 둘러싼 외부 환경을 지각하고 데이터를 수집하는 역할을 합니다.
(예: 카메라, 마이크, 온도 감지기)

핵심!

② 지능 (소프트웨어)

"사람의 두뇌"

센서로 수집한 정보를 분석하고, 어떻게 행동할지 학습하고 추론하여 결정합니다.

③ 액추에이터 (Actuator)

"사람의 팔과 다리"

지능의 명령을 받아 실제로 환경에 물리적인 변화를 일으키는 장치입니다.
(예: 로봇 팔, 스피커, 바퀴, 화면 표시)

헷갈리지 마세요! (오개념 방지)

"자동문도 사람이 오면 열리는데, 지능 에이전트인가요?"
자동문은 단순한 '센서'와 모터(액추에이터)만 있습니다. 하지만 상황을 보고 스스로 판단(누구인지, 열어줘야 할지 말지)하는 '지능(소프트웨어)' 요소가 없기 때문에 단순한 기계일 뿐, 지능 에이전트는 아닙니다.

로봇 청소기로 보는 에이전트의 3가지 특징

지능 에이전트는 일반 프로그램과 다르게 3가지 특별한 능력이 있습니다.

① 자율성

사람이 매번 리모컨으로 조종하지 않아도 됩니다.

로봇 청소기: "바닥의 먼지를 스스로 인식하고 알아서 청소하러 이동합니다."

② 협력성

다른 에이전트나 기계들과 통신하며 정보를 나눕니다.

로봇 청소기: "거실 청소기 1호와 방 청소기 2호가 서로 어디를 청소했는지 정보를 공유합니다."

③ 목표 지향성

주어진 '목표'를 달성하기 위해 최선의 방법을 찾습니다.

로봇 청소기: "'방 전체를 깨끗하게 만든다'는 목표 달성 여부가 에이전트를 평가하는 기준이 됩니다."

정리하자면! 지능 에이전트는 목적(목표 지향성)을 가지고, 스스로 판단하며(자율성), 필요할 땐 남과 소통(협력성)하는 스마트한 시스템입니다.

로봇 청소기 에이전트의 머릿속

로봇 청소기가 어떻게 자율적으로 판단하며 목표를 달성하는지 간단한 코드로 살펴봅니다.

로봇 청소기는 단순히 버튼을 누르면 동작하는 기계가 아니라, 목표가 달성될 때까지 스스로 상황을 판단하며 행동하는 지능 에이전트입니다.

핵심 아이디어
"방이 깨끗해질 때까지" 상태를 확인하면서 행동을 반복합니다.
robot_agent.c 

// 지능 에이전트(로봇 청소기)의 행동 논리

void startCleaningAgent() {
    // 목표 지향성: '방이 깨끗해질 때'까지 반복합니다.
    while (checkRoomStatus() != CLEAN) {

        // 자율성 1: 스스로 배터리를 확인합니다.
        if (getBatteryLevel() < 10) {
            moveToCharger(); // 충전소로 이동
            continue;
        }

        // 자율성 2: 카메라/센서로 먼지 위치를 스스로 파악합니다.
        Location dirtLoc = findDirt();

        // 파악한 위치로 이동하여 청소합니다.
        moveTo(dirtLoc);
        cleanArea();

        // 협력성: 다른 로봇에게 "여긴 내가 청소했어"라고 알립니다.
        sendMessageToOtherRobot("구역 A 청소 완료");
    }

    print("목표 달성! 청소를 종료합니다.");
}
